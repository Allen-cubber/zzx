# 📖 政务文本智能审核助手  - 使用手册

**【软件简介】**

本软件专为政务新媒体、公文写作及新闻稿件审核量身打造。采用“**本地专家词库 + DeepSeek 大语言模型”双擎驱动**”，能实现秒级文本纠错，精准揪出会议届次错误、机构名称缩写错误、过期口号及领导职务等涉政差错，为您的稿件安全保驾护航。

---

### 第一步：首次启动与基础配置

首次打开软件后，需要进行一次简单的初始化配置。配置会自动保存在您电脑的本地，后续无需重复操作。

1. **进入配置中心：** 点击软件左上角的 **“⚙️ 系统设置”** 按钮，唤出配置面板。
    
2. **填写大模型API-key：**  在第一栏“DeepSeek API 密钥”中，粘贴您的专用 API Key（通常以 `sk-` 开头）。
    
    - _注意：如果没有此密钥，软件将无法进行深度语义分析。_
        
3. **管理本地强控词库：**
    
    - 在第二栏的表格中，您可以管理本单位专属的“拦截黑名单”。
        
    - **添加规则：** 在下方输入框填入“错误原文”（如：_十八大_）和“正确表述”（如：_党的十八大_），点击 **“➕ 添加”**。
        
    - **删除规则：** 选中表格中的词条，点击右侧的 **“删除”**。
        
4. **保存生效：** 设置完毕后，务必点击底部的 **“💾 保存并生效”**。
    

---

### 第二步：核心操作流程

1. **粘贴稿件：**  将您需要审核的纯文本（如 Word、网页中复制的内容）直接粘贴到软件左侧巨大的空白“智能审校工作台”内。
    
    - _贴心提示：软件内置了格式净化功能，粘贴时会自动剥离乱七八糟的乱码格式。_
        
2. **一键体检：**  点击右上角的 **“🚀 DeepSeek 扫描”** 按钮。
    
    - 此时右侧会显示“⏳ AI 引擎深度思考中...”，请耐心等待几秒钟（时间长短取决于您的文本长度）。
        

---

### 第三步：沉浸式审阅与一键替换

扫描完成后，系统将进入“双屏联动”的校对模式：

- **🚨 发现错误（红框报警）：**
    
    - **左侧文章：** 所有疑似错误的地方，都会被标上**红色背景和红色下划线**。
        
    - **右侧侧边栏：** 会自动生成一张张“错题卡片”，清晰标注“❌ 原文”、“✅ 建议”以及“依据/原因”。
        
- **✅ 采纳建议（一键替换）：**
    
    - 浏览右侧卡片，如果您认可 AI 的修改建议，直接点击卡片下方的 **“✅ 替换”**。
        
    - 见证奇迹：点击后，左侧文章中的错词会**瞬间被替换为正确的词汇，并打上护眼的绿色高亮**，表示该风险已解除。
        
- **⏭️ 忽略建议（放行误报）：**
    
    - 如果您认为原文没问题（AI 偶尔会过于严格），只需点击卡片下方的 **“⏭️ 忽略”**。左侧的红色报警就会消除，恢复正常文本样式。
        

---

### 第四步：完成审核与导出

1. 当右侧边栏提示 **“🎉 太棒了！未发现明显政治性错误，文本安全可发”** 时，说明全篇已审核通过。
    
2. 将左侧工作台中所有变成绿色的（或正常的）最终安全文本全选（Ctrl+A / Cmd+A）并复制（Ctrl+C / Cmd+C）。
    
3. 粘贴回您的 Word 文稿或公众号后台发布即可。
    

---

> **💡 日常维护小贴士**
> 
> 如果有最新出台的会议口径，建议您第一时间打开 **“⚙️ 系统设置”**，将其加入“本地强制纠错词库”。本地词库享有最高优先级，且无需经过网络即可实现毫秒级拦截！

# 附录 如何获取 API Key？

## 🛠️ 第一步：注册 DeepSeek 账号

1. **访问官网**：
    
    打开浏览器，进入 [DeepSeek 官方开放平台](https://platform.deepseek.com/)。https://platform.deepseek.com/
    
2. **选择注册方式**：
    
    - 你可以选择使用 **手机号** 注册，或者关联 **微信** 账号快速登录。
        
3. **完成验证**：
    
    按照提示输入验证码并设置密码，确保账号安全。

---

## 🔑 第二步：创建 API Key
![[Pastedimage20260408214941.png]]
1. **进入控制台**：
    
    登录成功后，你会进入后台管理界面。
    
2. **找到 API Keys 选项**：
    
    在左侧导航栏中，点击 **“API Keys”** 菜单。
    
3. **创建新密钥**：
    
    - 点击页面中的 **“创建 api key”** 按钮。
        
    - **命名密钥**：为你的密钥起个名字（例如“我的测试”或“应用 A”），以便日后管理。
        
4. **保存密钥（至关重要）**：
    
    - 系统会弹出一个包含一串字符的窗口。
        
    - **注意**：API Key 仅在创建时显示**一次**。请立即将其复制并保存到安全的本地文件或密码管理器中。如果你关闭了窗口却没保存，就只能删除它重新创建一个了。
        

---

## 💰 第三步：充值与余额检查

DeepSeek 采用“先充值后使用”的模式（按 Token 计费）。

1. **检查余额**：
    
    点击左侧菜单的 **“充值”** 。
    
2. **充值**：
    
    - 新用户通常会有一定的**免费额度**，可以先试用。
        
    - 如果额度用完，选择金额并支付。DeepSeek 目前支持 **支付宝/微信支付**，非常方便。
        

---

- **查看文档**：详细的 API 参数和限制，可以随时查阅官方的 [API Documentation](https://api-docs.deepseek.com/)。
