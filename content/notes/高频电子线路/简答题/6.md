## 6.11
这是一道经典的通信电子线路分析题，考察的是对改进型高频LC振荡器（结合了Colpitts和Seiler/Clapp电路特点）的综合理解。

以下为您详细拆解该电路的各个部分：

### 1) 各元件的主要作用

- **晶体管 (V):** 电路的核心放大器件，提供维持等幅振荡所需的能量。
    
- **$C_1$、 $C_2$:** 反馈分压电容。它们构成了经典的电容三点式（Colpitts）反馈网络，将谐振槽路的一部分能量反馈到晶体管的基极-发射极之间，满足振荡的相位和幅度条件。
    
- **$L$、$C_7$、$C_6$、$C_5$:** 共同构成主谐振槽路，决定了主振荡器的基本工作频率。其中 $C_5$ 和 $C_7$ 为可变电容，用于调节频率。
    
- **$C_3$、 $C_4$:** 温度补偿电容。题目特别注明它们是“不同温度系数”的电容。将它们并联使用，可以通过正负温度系数的互相抵消，消除因温度变化引起的电容漂移，从而提高振荡器的频率稳定性。
    
- **$R_1$、 $R_2$:** 基极偏置分压电阻，为晶体管基极提供稳定的直流静态工作点。
    
- **$R_4$:** 发射极直流负反馈电阻，用于稳定晶体管的直流静态工作点，防止温漂。
    
- **$C_8$:** 发射极高频旁路电容。在工作频率下相当于短路，使高频交流信号无损耗地通过，避免被 $R_4$ 消耗。
    
- **$R_3$:** 基极防振电阻（阻尼电阻，阻值为 **200 Ω**）。它串联在基极回路中，可以降低寄生高频回路的Q值，有效抑制可能产生的甚高频（VHF）寄生振荡。
    
- **$L_2$:** 高频扼流圈（RFC）。隔离高频交流信号，防止振荡能量泄漏到直流电源 $V_{EE}$ 中。
    
- **$L_1$:** 发射极高频扼流圈。由于 $L_2 \gg L_1$，且 $L_1$ 仅为 **0.3 μH**，它在这里为发射极提供直流通道，同时在高频下呈现较高阻抗，防止交流信号通过电源回路直接对地短路。
    
- **$C_\phi$:** 电源低通滤波/旁路电容，确保电源端对交流信号呈现良好接地。

### 2) 画出交流等效电路

绘制交流等效电路需要遵循两个基本原则：**直流电源视为交流接地（即 $V_{EE}$ 端接地）**；**大容量旁路电容（$C_8$、$C_\phi$）视为短路**。通常高频扼流圈（$L_2$）视为开路。

由于我无法直接在屏幕上为您作图，请按照以下文字描述和拓扑关系在纸上绘制，这正是标准的交流等效模型：

1. **确定组态：** 晶体管的集电极（上方）直接接地，因此这是一个**共集电极（CC）组态**。
    
2. **发射极连接：** 晶体管的发射极（E）通过电容 $C_1$ 直接连接到地。
    
3. **基极与反馈连接：** 晶体管的基极（B）首先串联电阻 $R_3$ 到达一个节点（设为节点 D）。节点 D 与发射极（E）之间跨接反馈电容 $C_2$。节点 D 同时通过并联后的偏置电阻 $(R_1 // R_2)$ 接地。
    
4. **谐振槽路连接：** 节点 D 也是主谐振槽路的入口。它连接到并联电容 $(C_3 // C_4)$，然后串联接入后面的槽路网络：即先遇到对地并联的可变电容 $C_5$，再串联耦合电容 $C_6$，最后连接到并联的谐振对 $(L // C_7)$，该谐振对的另一端接地。
    

_(简而言之：集电极接地；$C_1$ 接在发射极与地之间；$C_2$ 接在基极电阻 $R_3$ 之后与发射极之间；整个庞大的LC网络接在 $R_3$ 之后与地之间。)_

### 3) 分析该电路的特点

这是一个设计非常严谨的高稳定度调幅通信机主振荡器，其核心特点集中在“高频率稳定性”上：

1. **采用了弱耦合结构（类似西勒 Seiler 电路特性）：** 主谐振回路（$L$、$C_7$）并不是直接并联在晶体管两端，而是通过较小的电容（$C_6$ 以及 $C_3 // C_4$）串联耦合接入晶体管。这种“弱耦合”极大地减小了晶体管内部结电容（随温度和电压变化而波动）对主谐振回路频率的影响，提高了频率的长期稳定性。
    
2. **共集电极（CC）组态的优势：** 电路交流等效为共集组态。共集电极电路具有**输入阻抗极高**的特点。既然谐振槽路接在基极侧，高输入阻抗就意味着晶体管对谐振槽路的“负载效应”非常轻，从而维持了槽路的**高Q值**。
    
3. **多重物理级稳频措施：**
    
    - **温度对冲：** 巧妙地利用了具有相反温度系数的 $C_3$ 和 $C_4$ 进行并联，直接在物理层面抵消了温度造成的电容漂移。
        
    - **抑制寄生干扰：** $R_3$ 的加入破坏了寄生振荡的起振条件，保证了输出频谱的纯净，这在通信机的主振器中至关重要。
        
4. **降低负载牵引效应：** 观察输出端可以发现，信号是从电感 $L$ 的抽头（或次级线圈）引出的，而不是直接从槽路两端引出。这减小了后级负载阻抗变化对振荡器频率的“牵引”作用。


**结论：会引起频率发生变化。**

这种现象在通信电子线路中通常被称为“调幅-调频转换”（AM-FM 转换效应）**或**幅度-频率牵引效应。

振幅不稳定之所以会牵连频率发生变化，主要有以下三个核心原因：

## 6.13

### 1. 有源器件（晶体管）的非线性电容效应（最主要原因）

晶体管（无论是双极型三极管 BJT 还是场效应管 MOSFET）内部都存在**结电容**（例如 BJT 的发射结电容 $C_{be}$、集电结电容 $C_{bc}$，或 MOS 管的栅源电容 $C_{gs}$）。

- 这些结电容不是恒定不变的，而是**非线性的势垒电容**，其容量大小强烈依赖于施加在 PN 结两端的电压。
    
- 当振荡器的振幅发生变化时，晶体管极间的电压摆幅和平均直流电位也会随之改变，这就导致**结电容的等效平均电容量发生变化**。
    
- 由于晶体管的结电容实际上是部分并联或等效接入主谐振回路（LC 槽路）的，结电容的改变直接改变了回路的总电容 $C$，根据谐振频率公式 $f_0 = \frac{1}{2\pi\sqrt{LC}}$，振荡频率必然随之发生偏移。
    

### 2. 非线性相移与谐波效应

振荡器通常工作在非线性状态（如丙类）。振荡幅度不稳定意味着晶体管的导通角 $\theta_c$ 和非线性工作深度在发生波动。

- 这会导致输出电流脉冲的形状发生改变，进而使电流中的**谐波成分（二次、三次谐波等）比例发生变化**。
    
- 虽然主谐振回路对基波谐振，但高次谐波在回路中流动时，会通过反馈网络对基波产生额外的**附加相移**。
    
- 根据上一题我们分析过的“相位平衡条件”（$\Sigma \phi = 0$），为了抵消这个由振幅波动引起的附加相移，谐振回路必须“被迫”偏离原本的中心频率，以产生一个反向的相移来维持振荡，这就导致了频率的变化。
    

### 3. 功耗与温度效应（热漂移）

虽然这是一个较慢的物理过程，但也不容忽视：

- 振幅的波动意味着电路中产生和消耗的功率在波动。振幅增大时，晶体管的耗散功率增加，线圈和电阻上的热损耗也增加。
    
- 功耗的波动会导致**局部温度的波动**。
    
- 谐振回路中的电感线圈和电容器件都有各自的温度系数，温度变化会引起元器件物理形态或介电常数的微小改变，最终表现为频率的温漂。

## 6.14

这是一道考察振荡器**实际工作状态**与**理想模型**之间差异的核心概念题。结合前面我们讨论过的相位平衡条件，这个问题可以迎刃而解。

以下是详细的解答：

### 1. 为什么工作频率（$f_{osc}$）一般总是不等于自然谐振频率（$f_0$）？

**核心原因：为了强行满足振荡器的“相位平衡条件”。**

- **理想情况：** 我们在简化分析时，通常假设放大器（晶体管）和反馈网络不产生额外相移（即 $\varphi_Y + \varphi_F = 0^\circ$）。此时，为了满足总相移为 0 的条件，谐振回路必须呈现纯阻性，相移 $\varphi_Z = 0^\circ$。这个时候，振荡频率恰好等于回路的自然谐振频率 $f_0$。
    
- **实际情况：** 在高频工作状态下，晶体管内部存在**结电容**和载流子的**渡越时间效应**，分布参数也会产生影响。这就导致放大器和反馈网络不可避免地会引入一个寄生相移（即 $\varphi_Y + \varphi_F \neq 0$）。
    
- **被迫失谐：** 根据相位平衡条件 $\Sigma \varphi = \varphi_Y + \varphi_F + \varphi_Z = 0$，为了抵消掉这部分额外的寄生相移，谐振网络**必须**产生一个大小相等、符号相反的补偿相移 $\varphi_Z = -(\varphi_Y + \varphi_F)$。
    
- **结论：** 谐振回路只有在**偏离**其自然谐振频率 $f_0$ 时（呈现容性或感性），才能产生所需的相移。因此，实际的振荡频率 $f_{osc}$ 一般总是不等于 $f_0$。
    

---

### 2. 失谐量大小与哪些因素有关？

失谐量（即 $\Delta f = f_{osc} - f_0$ 或 $\Delta \omega$）的大小主要取决于以下两个核心因素：

#### 因素一：外部引入的附加相移大小（需求量）

失谐的初衷是为了补偿 $\varphi_Y$ 和 $\varphi_F$。因此，如果外部引入的相移越大，回路需要“憋出”的补偿相移 $\varphi_Z$ 就越大。

- 晶体管的结电容受**电源电压波动**、**温度漂移**、**器件老化**或**振幅变化**（即上一题提到的调幅-调频效应）影响越大，寄生相移的变化就越剧烈，导致的失谐量也就越大。
    

#### 因素二：谐振回路的品质因数 $Q$（供给能力）

这决定了回路产生相移的“效率”。我们在前面的相频特性曲线图中得出过结论：

- **$Q$ 值越高：** 谐振回路的相频特性曲线越**陡峭**。这意味着，只需要极微小的一点点频率偏移量 $\Delta f$，回路就能产生很大的相移来满足补偿需求。此时**失谐量极小**，频率稳定性极高（比如石英晶体振荡器）。
    
- **$Q$ 值越低：** 曲线越**平缓**。为了获得同样的补偿相移，频率必须大幅度偏离中心位置。此时**失谐量很大**，频率极易受外界干扰而飘移。
