# 填空题
## 1. 考察串并联谐振
这是一道典型的高频电子线路（并联谐振回路）题目。以下是详细的解答步骤：

### **已知参数**

- 谐振频率：$f_0 = 5\text{ MHz} = 5 \times 10^6\text{ Hz}$
    
- 回路电容：$C = 50\text{ pF} = 50 \times 10^{-12}\text{ F}$
    
- 初始通频带：$BW_1 = 2\Delta f_{0.7} = 150\text{ kHz} = 150 \times 10^3\text{ Hz}$
    

### **第一空：计算电感 L**

并联谐振回路的谐振频率公式为：

$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

对其两边平方并变形，可求得电感 $L$：

$$L = \frac{1}{4\pi^2 f_0^2 C}$$

代入已知数据：

$$L = \frac{1}{4\pi^2 \times (5 \times 10^6)^2 \times 50 \times 10^{-12}} = \frac{1}{5000\pi^2}\text{ H}$$

将其换算为微亨（$\mu\text{H}$）并保留两位小数：

$$L \approx 20.26\text{ }\mu\text{H}$$

### **第二空：计算需要并联的电阻阻值**

对于并联谐振回路，其通频带 $BW$ 与回路等效并联电阻 $R_p$ 和电容 $C$ 满足以下关系式：

$$BW = \frac{1}{2\pi R_p C}$$

由该公式可知，初始状态下回路的等效并联电阻 $R_1$ 为：

$$R_1 = \frac{1}{2\pi \cdot BW_1 \cdot C} = \frac{1}{2\pi \times 150 \times 10^3 \times 50 \times 10^{-12}} = \frac{10^6}{15\pi}\text{ }\Omega$$

题目要求将通频带增加到 $BW_2 = 300\text{ kHz}$。由于通频带扩大为原来的 **2 倍**，且通频带与等效并联电阻成反比，这意味着新的总等效并联电阻 $R_2$ 必须是原来的 **二分之一**：

$$R_2 = \frac{R_1}{2}$$

为了让电路的总等效并联电阻精确减半，根据电阻并联分流原理（$\frac{1}{R_2} = \frac{1}{R_1} + \frac{1}{R_x}$），我们需要在回路两端并联一个与原等效电阻 $R_1$ **阻值完全相同** 的电阻 $R_x$。

因此，应并联的电阻阻值为：

$$R_x = R_1 = \frac{10^6}{15\pi}\text{ }\Omega$$

计算近似值：

$$R_x \approx 21220.66\text{ }\Omega \approx 21.22\text{ k}\Omega$$

### **最终答案填空**

1. 电感 L 的值为：**$\frac{1}{5000\pi^2}\text{ H}$** （或填 **$20.26\text{ }\mu\text{H}$**）
    
2. 回路两端应并联的电阻阻值为：**$\frac{10^6}{15\pi}\text{ }\Omega$** （或填 **$21.22\text{ k}\Omega$**）

## 4. 品质因数

1. 空载品质因数：**越大越好**（或填 **大**）
    
2. 有载品质因数：**越小越好**（或填 **小**）
### **原理解析**

在谐振功率放大器的输出匹配网络中，网络的传输效率 $\eta_K$（即传递给实际负载的功率与输入到匹配网络的总功率之比）与空载品质因数 $Q_0$ 和有载品质因数 $Q_L$ 之间满足以下关系式：

$$\eta_K = 1 - \frac{Q_L}{Q_0}$$

根据该公式可以得出以下结论：

- **空载品质因数 $Q_0$**：反映了谐振回路元件（如电感、电容）自身的固有损耗。为了减少网络内部的热损耗、提高传输效率，元件的质量要好，即 **$Q_0$ 应该尽可能大**。
    
- **有载品质因数 $Q_L$**：反映了接入负载后回路的整体损耗情况。由公式可知，为了使 $\eta_K$ 趋近于 1（即 100%），比值 $\frac{Q_L}{Q_0}$ 必须尽量小，因此 **$Q_L$ 应该尽可能小**。

## 6. 功率放大器工作状态与频率的关系

功率放大器工作于临界状态，若激励电压的频率提高一倍，幅度不变，则负载功率 **减小（下降）**，工作状态变为 **欠压** 状态。

### **原理解析**

此题考察高频谐振功率放大器中**负载谐振回路**的频率特性与**工作状态**之间的关系。

**1. 谐振功放的负载特性回顾**

高频谐振功率放大器（丙类）的负载为 **LC 并联谐振回路**，该回路调谐于输入信号的基波频率 $f_0$。根据笔记[[../高频电子线路/index.md|高频电子线路笔记]]第5章内容：

> LC并联谐振回路对**基波** ($f_0 = \omega/2\pi$) 谐振，呈现**纯阻** $R_p$ (阻值很大)。对各**次谐波**失谐，阻抗很低 (呈容性或感性)。

三种工作状态由等效负载 $R_p$ 的大小决定（笔记[[../复习总结/index.md|复习总结]]）：

| 状态 | 负载 $R_p$ | 特点 |
| :--- | :--- | :--- |
| **欠压** | 较小 | $I_{cm1}$ 基本恒定（近似恒流源），$P_o$ 随 $R_p$ 增大而增大 |
| **临界** | 合适 | 输出功率**最大**，效率较高（最佳工作状态） |
| **过压** | 较大 | 出现凹陷，输出电压平坦，效率高但波形差 |

**2. 频率提高一倍的影响**

当激励电压频率从 $f_0$ 提高到 $2f_0$（提高一倍）时：

- LC 并联回路原本调谐于 $f_0$，在 $2f_0$ 处严重**失谐**，其等效阻抗急剧下降。
- 这等效于晶体管集电极看到的等效负载电阻 $R_p$ **大幅减小**。
- 根据负载特性：$R_p$ 减小 → 放大器从**临界状态**进入**欠压状态**。
![图片](Pasted-image-20250417210720.png)
**3. 负载功率的变化**

由笔记可知，基波输出功率公式为：

$$P_o = \frac{1}{2} I_{cm1}^2 R_p = \frac{1}{2} \frac{U_{cm}^2}{R_p}$$

在欠压区，$I_{cm1}$ 近似恒定（恒流源特性），但由于 $R_p$ 急剧减小，输出功率 $P_o$ 随之**显著减小**。

同时，由于回路对 $2f_0$ 失谐，集电极电流脉冲中的基波分量（$f_0$）无法在负载上建立有效电压，而 $2f_0$ 分量（二次谐波）虽能通过但幅度较小且不是回路设计的谐振点，因此负载上获得的功率**大幅下降**。

### **最终答案填空**

- 负载功率：**减小**（或填**下降**）
- 工作状态变为：**欠压**状态

## 7. 石英晶体振荡器的工作频率与类型

石英晶体的串联谐振频率为 $f_q$，并联谐振频率为 $f_p$。在石英晶体振荡器中，晶体作为电感元件使用时，振荡器的工作频率为 **$f_q < f < f_p$（介于 $f_q$ 与 $f_p$ 之间）**，这种振荡器称为 **并联型晶体** 振荡器（典型电路为 **皮尔斯(Pierce)** 振荡器）。

### **原理解析**

**1. 石英晶体的阻抗频率特性**

根据笔记[[../高频电子线路/index.md|高频电子线路笔记]]第6章，石英晶体的等效电路及阻抗特性如下：

| 频率范围 | 阻抗性质 | 说明 |
| :--- | :--- | :--- |
| $\omega < \omega_s$（$f < f_q$） | **容性** | — |
| $\omega_s < \omega < \omega_p$（$f_q < f < f_p$） | **感性** | ⚠️ 晶体作为**电感**使用的**唯一区域**！ |
| $\omega > \omega_p$（$f > f_p$） | **容性** | — |

> $\omega_s < \omega < \omega_p$：呈感性 (+jX)。⚠️ 这是晶体作为**电感**使用的**唯一区域**！

晶体在 $f_q \sim f_p$ 之间呈感性，且此区间非常窄（通常相差几 kHz 到几十 kHz），阻抗相位随频率变化**极其陡峭**，这正是晶体振荡器具有极高频率稳定度的根源。

**2. 两种晶体振荡器类型对比**

| 类型 | 晶体作用 | 工作频率 | 典型电路 |
| :--- | :--- | :--- | :--- |
| **并联型** | 等效**电感** | $f_q < f < f_p$ | **皮尔斯(Pierce)**、密勒(Miller) |
| **串联型** | 等效**短路**（低阻抗） | $f \approx f_q$ | 串联型晶体振荡器 |

**3. 并联型晶体振荡器（重点）**

当晶体作为电感元件使用时：
- 利用晶体在 $f_q < f < f_p$ 范围内呈**感性**的特性
- 将晶体作为**等效电感**接入振荡电路（替代 LC 振荡器中的 L）
- 振荡频率 $f$ 满足 $f_q < f < f_p$

**最典型的并联型电路 — 皮尔斯(Pierce)振荡器**：
- 晶体接在**集电极（或漏极）与基极（或栅极）**之间
- 等效于**考毕兹(Colpitts)**电路，晶体充当电感 L
- $C_1, C_2$ 既是谐振回路的一部分，也构成反馈分压网络

此外还有**密勒(Miller)振荡器**，晶体接在基极与发射极之间，等效于哈特莱电路，但其频率稳定度不如皮尔斯电路。

**4. 串联型晶体振荡器**

晶体在 $f \approx f_q$ 时阻抗最小（近似短路），串联接入正反馈环路中，仅在 $f_q$ 处满足振荡条件。此时晶体相当于一个**选频短路元件**。

### **最终答案填空**

- 振荡器的工作频率为：**$f_q < f < f_p$**（或填**介于 $f_q$ 与 $f_p$ 之间**）
- 这种振荡器称为：**并联型晶体** 振荡器（或填**皮尔斯(Pierce)** 振荡器）

## 8. 调幅广播的频带宽度

调幅广播，广东电台的载波频率为1035kHz，音频调制信号频率为90Hz-5.5kHz，则此电台信号所占的频带宽度为 **11 kHz**。

### **原理解析**

此题考察**普通调幅波（AM）频带宽度**的计算，属于第7章振幅调制的基础考点。

**1. AM 调幅波的频谱结构**

根据笔记[[../复习总结/index.md|复习总结]]第7章：

单音频调制的调幅波包含**三个频率分量**：载波分量（$f_c$）、上边频分量（$f_c + F$）和下边频分量（$f_c - F$）。当调制信号为多音频（实际语音/音乐信号，频率范围为 $F_{min} \sim F_{max}$）时，会产生**上边带**和**下边带**。

| 调制方式 | 频带宽度 |
| :--- | :--- |
| **AM（普通调幅）** | $BW = 2F_{max}$ |
| DSB-SC（抑制载波双边带） | $BW = 2F_{max}$ |
| SSB（单边带） | $BW = F_{max}$ |

**2. 计算公式**

调幅波的频带宽度仅由**调制信号的最高频率**决定，与载波频率无关：

$$\boxed{BW = 2F_{max}}$$

**3. 代入计算**

- 载波频率：$f_c = 1035\text{ kHz}$（仅决定信号在频谱中的位置，不影响带宽）
- 音频调制信号频率范围：$90\text{ Hz} \sim 5.5\text{ kHz}$
- 最高调制频率：$F_{max} = 5.5\text{ kHz}$

$$BW = 2 \times F_{max} = 2 \times 5.5 = 11\text{ kHz}$$

**4. 频谱示意**

```
      下边带         载波         上边带
    ←── 5.5kHz →     │     ←── 5.5kHz →
    ┌────────┐       │       ┌────────┐
    │        │       │       │        │
────┴────────┴───────┴───────┴────────┴────→ f
  1029.5   1035   1035   1035   1040.5   (kHz)
```

该电台信号占据了从 $1029.5\text{ kHz}$ 到 $1040.5\text{ kHz}$ 的频率范围，总带宽为 $11\text{ kHz}$。

### **最终答案填空**

- 此电台信号所占的频带宽度为：**11 kHz**

## 1. 晶体管的单向化问题（6分）

**题目**：为什么晶体管在高频工作时需要考虑单向化问题，而在低频工作时，则不需要考虑。有哪几种单向化的方法？

### **答题要点**

**（一）高频需单向化，低频不需要（3分）**

晶体管内部存在 $C_{b'c}$（集电结电容/密勒电容），构成内部反馈（y参数中用 $y_{re}$ 表示）。
- 高频时 $C_{b'c}$ 容抗 $=1/(\omega C_{b'c})$ 随频率升高而减小 → 内部反馈增大 → 输入输出相互影响、调谐困难，严重时自激振荡。
- 低频时容抗极大，内部反馈可忽略。

**（二）单向化方法（3分）**

① **失配法**：故意使阻抗失配以减小反馈 → 典型电路：共射-共基级联（Cascode）。
② **中和法**：外加反相反馈电路抵消内部反馈 → 外加中和电容 $C_N$。

---

### **📝 卷面答案**

> **答：**
> （1）晶体管集电结电容 $C_{b'c}$（密勒电容）构成内部反馈。高频时其容抗减小，反馈增大，导致输入输出相互影响、调谐困难，甚至自激振荡；低频时容抗很大，反馈可忽略，故无需单向化。**【3分：$C_{b'c}$/$y_{re}$内部反馈1分 + 高频影响1分 + 低频可忽略1分】**
>
> （2）两种方法：① **失配法**——故意使阻抗失配减小反馈，典型电路为共射-共基级联（Cascode）；② **中和法**——外加反反馈电路，产生与内部反馈大小相等、极性相反的信号相抵消，通过外加中和电容 $C_N$ 实现。**【3分：两种方法名各1分 + 简要原理/典型电路1分】**

## 2. 变频器的功能与实现原理（6分）

**题目**：变频器的功能是什么？用什么样的器件能实现变频的作用，为什么？

### **答题要点**

**（一）功能**：将输入信号频谱沿频率轴**不失真搬移**（频谱线性搬移），即改变载波频率而保持包络不变。$f_I = |\pm f_o \pm f_s|$，接收机中常用 $f_I = f_o - f_s$（下变频）。

**（二）器件**：**非线性器件**（二极管、晶体管等）或乘法器。

**（三）原因**：非线性器件的伏安特性用幂级数展开为 $i = a_0 + a_1 v + a_2 v^2 + \cdots$，其中 $a_2 v^2$ 产生**乘积项**，积化和差后产生和频与差频（新频率分量）。线性器件满足叠加定理，不能产生新频率，故无法实现变频。

---

### **📝 卷面答案**

> **答：**
> （1）变频器的功能是将输入已调信号的频谱沿频率轴进行不失真搬移（频谱线性搬移），把载波频率 $f_s$ 变换为固定中频 $f_I = |\pm f_o \pm f_s|$，而调制包络保持不变。**【2分】**
>
> （2）采用非线性器件（二极管、晶体管等）或线性时变器件实现。**【2分】**
>
> （3）变频需要产生新的频率分量（和频与差频）。非线性器件的伏安特性展开为幂级数 $i = a_0 + a_1v + a_2v^2 + \cdots$，其平方项产生两信号的乘积项，经积化和差可得和频与差频。而线性器件满足叠加定理，不能产生新频率，故无法实现变频。**【2分：幂级数/平方项/乘积项1分 + 线性不能产生新频率1分】**

## 3. 调幅收音机接收调频广播的原理（6分）

**题目**：假设调幅收音机的工作波段与调频广播的波段相同，那么只要改变调幅收音机的本振频率，其它基本不变，即可实现用调幅收音机接收调频广播，请分析原因。

### **答题要点**

核心原理：**斜率鉴频**（失谐鉴频）——利用谐振回路幅频特性斜率，将 FM → AM-FM → 包络检波。

- AM收音机中放含 LC 谐振回路，正常接收AM时载波对准**中心（峰值）**。
- 改变本振频率后，FM中频信号中心频率落到谐振曲线**斜坡**（斜率最大处）。
- FM频率变化 → 回路增益随之变化 → 输出幅度按频率规律变化 → FM 变为 **AM-FM 波**。
- AM收音机原有的**包络检波器**检出幅度变化 → 恢复调制信号。

---

### **📝 卷面答案**

> **答：**
> 利用**斜率鉴频（失谐鉴频）**原理实现：
>
> （1）AM收音机中频放大器采用LC谐振回路，其幅频特性为钟形曲线，正常接收AM时载波频率对准曲线中心（增益最大处）。**【2分：谐振曲线/失谐概念】**
>
> （2）改变本振频率后，FM信号经混频得到的中频信号中心频率不再对准中心，而是落在曲线一侧的**斜坡**（近似线性段）上。此时FM信号的频率变化 $\Delta f$ 引起谐振回路输出幅度按斜坡斜率相应变化 $\Delta A$，从而将FM波转换为**AM-FM波**（幅度随瞬时频率变化）。**【2分：频率→幅度转换机制】**
>
> （3）AM收音机原有的**包络检波器**（二极管+RC网络）对AM-FM波进行检波，检出幅度变化分量，即可恢复出原始调制信号。**【2分：包络检波完成解调】**

## 4. 单边带信号产生方法

#### **1. 放大器输出端**

- **作用：** 对输入的低频调制信号进行电压/功率放大。放大器是线性器件，不会改变信号的频率成分。
    
- **频谱特征：** 频谱形状与原调制信号完全一致。
    
- **频率范围：** $300\text{ Hz} \sim 3000\text{ Hz}$（即 $0.3\text{ kHz} \sim 3\text{ kHz}$）。
    

#### **2. 平衡调制器输出端**

- **作用：** 产生双边带抑制载波（DSB-SC）信号。它将调制信号与频率为 $f_1 = 5\text{ MHz} = 5000\text{ kHz}$ 的本振信号相乘。
    
- **频谱特征：** 载波 $f_1$ 被抑制（通常画成虚线箭头）。原基带频谱被搬移到 $f_1$ 的两侧，形成上边带（USB）和下边带（LSB）。原频谱的形状在此处会相对于载波发生“镜像”对称。
    
- **频率范围：**
    
    - 下边带 (LSB)：$f_1 - 3\text{ kHz}$ 到 $f_1 - 0.3\text{ kHz}$ $\Rightarrow$ **$4997\text{ kHz} \sim 4999.7\text{ kHz}$**
        
    - 上边带 (USB)：$f_1 + 0.3\text{ kHz}$ 到 $f_1 + 3\text{ kHz}$ $\Rightarrow$ **$5000.3\text{ kHz} \sim 5003\text{ kHz}$**
        

#### **3. 上边带滤波器输出端**

- **作用：** 这是一个带通滤波器，目的是滤除下边带，只保留上边带信号。
    
- **频谱特征：** 仅剩下 $f_1$ 右侧的频谱分量。
    
- **频率范围：** **$5000.3\text{ kHz} \sim 5003\text{ kHz}$**
    

#### **4. 混频器输出端**

- **作用：** 将中频的单边带信号上变频到射频频段。混频器将上一步的输入信号与 $f_2 = 15\text{ MHz} = 15000\text{ kHz}$ 的本振信号进行混频，产生和频与差频。
    
- **频谱特征：** 理想混频器输出包含“本振+输入”和“本振-输入”两个频段的信号（注意：实际应用中往往只取和频，但纯粹的混频器输出端两者皆有）。
    
- **频率范围：**
    
    - 和频分量 ($f_2 + f_{USB}$)：$15000 + 5000.3 \sim 15000 + 5003$ $\Rightarrow$ **$20.0003\text{ MHz} \sim 20.003\text{ MHz}$**
        
    - 差频分量 ($f_2 - f_{USB}$)：$15000 - 5003 \sim 15000 - 5000.3$ $\Rightarrow$ **$9.997\text{ MHz} \sim 9.9997\text{ MHz}$**
        

#### **5. 线性功率放大器及天线网络输出端**

- **作用：** 放大射频信号以驱动天线。这里的功放和后续的 $L$、$C_1$、$C_2$ 匹配网络（本质是一个调谐回路）会起到带通滤波的作用，选出我们需要的最终发射频率（通常是和频）。
    
- **频谱特征：** 滤除了混频产生的差频分量和其他杂波，只保留最终的射频上边带信号。
    
- **频率范围：** **$20.0003\text{ MHz} \sim 20.003\text{ MHz}$**
    
# 电路分析题

## 1. 石英晶体振荡器

*   **晶体作用**:
    *   在 $\omega_s < \omega < \omega_p$ 时，作为<font color="orange">电感元件</font>使用 (用于并联型)。
    *   在 $\omega \approx \omega_s$ 时，作为<font color="orange">短路元件</font>或<font color="orange">低阻抗</font>元件使用 (用于串联型)。

- **基极-发射极 ($b-e$) 之间：** 接有 $200\text{ pF}$ 的电容，呈现 **容性**。
    
- **发射极-集电极 ($e-c$) 之间：** 接有 $300\text{ pF}$ 电容与 $47\text{ }\mu\text{H}$ 电感组成的并联回路。我们来计算该 $LC$ 回路的固有谐振频率 $f_{LC}$：
    
    $$f_{LC} = \frac{1}{2\pi\sqrt{LC}} = \frac{1}{2\pi\sqrt{47\times 10^{-6} \times 300\times 10^{-12}}} \approx 1.34\text{ MHz}$$
    
    因为标称振荡频率为 $5\text{ MHz}$，远高于该 $LC$ 回路的固有谐振频率（$5\text{ MHz} > 1.34\text{ MHz}$），所以该并联 $LC$ 回路在 $5\text{ MHz}$ 时呈现 **容性**。
    
- 此时与发射极相连的两个回路均呈容性，满足“射同”。为了满足起振的相位条件，**基极-集电极 ($b-c$) 之间** 的回路必须呈现 **感性**。
    
- $b-c$ 之间接有石英晶体 $J_T$ 和 $20\text{ pF}$ 电容。当石英晶体工作在其串联谐振频率 $f_s$ 和并联谐振频率 $f_p$ 之间时，它呈现出很大的等效电感。该等效电感与 $20\text{ pF}$ 电容串联后，整体对外呈现 **感性**，从而满足了三点式振荡器的起振条件。
	
- 根据“射同基反”原则，该电路是并联谐振型晶体振荡器电路或皮尔斯电路。电路满足电容三端振荡电路的组成原则。晶体起电感作用。当工作频率变为 $1\text{ MHz}$ 时，工作频率**低于**了并联回路的谐振频率，因此该 $LC$ 并联回路对外呈现 **感性**，破坏了三点式振荡器“射同基反”，不能振荡。

## 2. 超外差接收机前端
 
### （1）该电路的作用是什么？其输入信号有两个？分别从晶体管的哪一极注入？

- **电路的作用：** 这是一个**自激混频器**。它通常用于超外差接收机中，作用是同时产生本机振荡信号，并将其与接收到的高频调幅信号进行混频，从而将不同频率的高频信号统一变换为固定的中频信号。
    
- **两个输入信号及注入极：**
    
    1. **高频外来信号（射频信号）：** 由天线接收，经过 $L_A$ 与 $L_1$ 耦合，从晶体管的 **基极** 注入。
        
    2. **本机振荡信号（本振信号）：** 由晶体管自身及其发射极、集电极相关的 $LC$ 回路（$L_3, L_4, C_3, C_{1b}$ 等）自激振荡产生，等效于从晶体管的 **发射极** 注入参与混频。
### （2）三个并联谐振电路调谐在什么频率？它们的频率关系是怎样的？

- **三个调谐回路及其频率：**
    
    1. **输入调谐回路**（天线回路，$L_1$ 与 $C_{1a}, C_b$ 等组成）：调谐在接收的高频信号频率 **$f_s$**（或外来电台载波频率 $f_c$）。
        
    2. **本振调谐回路**（$L_4$ 与 $C_3, C_{1b}$ 等组成）：调谐在本机振荡频率 **$f_L$**（或 $f_o$）。
        
    3. **中频调谐回路**（集电极输出端的 $L_5$ 与 $220\text{ pF}$ 电容组成，即中频变压器/中周）：调谐在固定的中频频率 **$f_I$**（在我国的标准 AM 广播中，此频率通常为 $465\text{ kHz}$）。
        
- **频率关系：**
    
    为了通过混频得到固定的中频输出，本振频率必须始终比输入信号频率高出一个固定的中频值。其频率关系为：
    
    $$f_L - f_s = f_I$$
    

### （3）$C_{1a}$ 和 $C_{1b}$ 连接在一起调谐的原因是什么？

图中 $C_{1a}$ 和 $C_{1b}$ 用虚线连接在一起，代表它们是**同轴联动的双连可变电容器**。

- **连接调谐的原因（实现统调/同步调谐）：**
    
    在超外差接收机中，为了接收不同频率的电台信号，必须调整输入回路的谐振频率 $f_s$。根据上述 $f_L - f_s = f_I$ 的关系，为了保证混频后产生的中频信号 $f_I$ 始终准确且固定不变，本机振荡频率 $f_L$ 必须随着 $f_s$ 的改变而**同步改变**。
    
    使用双连电容可以使得用户在转动一个选台旋钮时，输入回路电容 $C_{1a}$ 和本振回路电容 $C_{1b}$ 同步发生相应的变化，从而保证两个回路的谐振频率差值始终保持为一个恒定的中频频率，这一过程在工程上称为“统调”。

## 3. 比例鉴频器

### （1）$C_6$ 的作用是什么？

$C_6$ 是一个大容量电容，其核心作用是**抑制寄生调幅（即限幅作用）**。

在比例鉴频器中，$C_6$ 与负载电阻（$R_3$、$R_4$等）构成了一个充放电时间常数非常大的回路。当输入调频信号的幅度发生快速波动（寄生调幅）时，由于 $C_6$ 两端的电压不能突变，它能够维持鉴频器总的直流负载电压基本不变，从而使得电路对输入信号的幅度变化不敏感，省去了前级的专用的限幅器电路。

### （2）在鉴频信号频率范围内，$L_3$ 的感抗和 $C_3$ 的容抗两者的模值比较哪一个更大，为什么？

**$L_3$ 的感抗模值远大于 $C_3$ 的容抗模值。**

**原因分析：**

- **$C_3$ 的作用：** 它是高频耦合电容，目的是将初级回路的高频电压（$\dot{V}_{12}$）尽可能无衰减地耦合到次级回路的中心抽头处。为了达到这个目的，在工作频率下，$C_3$ 的容抗必须非常小，对高频信号近似相当于**短路**。
    
- **$L_3$ 的作用：** 它是高频扼流圈（高频阻流圈）。它的作用是允许整流后的直流和低频（解调出的音频）信号顺利通过，同时阻止高频信号直接对地短路泄漏。因此，在工作频率下，$L_3$ 的感抗必须非常大，对高频信号近似相当于**开路**。
    

### （3）若两个二极管 $D_1$ 和 $D_2$、$C_6$ 的极性都倒过来，电路是否还能鉴频？鉴频特性如何变化。

**电路依然能够鉴频。**

**鉴频特性变化分析：**

当二极管 $D_1$ 和 $D_2$ 的极性反接后，整流电流的方向会改变，这会导致检波输出的直流电压极性全部翻转（这也是为什么题目提到将电解电容 $C_6$ 的极性也一并倒过来，以配合反向的直流电压）。

虽然检波电压极性反了，但初次级回路因为频率频偏引起相位差变化、进而导致两个二极管上的高频电压模值产生差值的**基本工作原理并没有改变**。

因此，电路仍然具备鉴频功能。其具体的鉴频特性变化是：**鉴频特性曲线（S型曲线）会发生上下翻转（即极性反相）**。原本频率升高时输出正电压的情况，会变成输出负电压，鉴频跨导的符号发生改变。
