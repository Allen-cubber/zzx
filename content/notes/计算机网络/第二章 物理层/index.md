在计算机网络体系结构中，<font color="orange">物理层</font>处于最底层（OSI模型的第1层，TCP/IP模型的网络接口层底部）。

## 2.1 物理层的基本概念 💡

*   **核心考察点**：物理层考虑的是怎样才能在连接各种计算机的传输媒体上传输数据<font color="orange">比特流</font>，而**不是**指具体的传输媒体。
*   **作用**：尽可能<font color="orange">屏蔽</font>掉不同传输媒体和通信手段的差异。
*   **四大特性**（确定与传输媒体的<font color="orange">接口</font>特性）：
    1.  **机械特性**：指明接口所用接线器的形状和尺寸、引线数目和排列、固定和锁定装置等。
    2.  **电气特性**：指明在接口电缆的各条线上出现的电压的范围。
    3.  **功能特性**：指明某条线上出现的某一电平的电压的意义。
    4.  **过程特性**：指明对于不同功能的各种可能事件的出现顺序。

## 2.2 数据通信的基础知识 📡

### 2.2.1 数据通信系统的模型
三大部分：**源系统**（发送端）、**传输系统**（传输网络）和**目的系统**（接收端）。
*   **常用术语**：
    *   **消息**：如话音、文字、图像、视频等。
    *   **数据**：运送消息的实体。有意义的符号序列。
    *   **信号**：数据的电气的或电磁的表现。
        *   **模拟信号**：代表消息的参数的取值是连续的。
        *   **数字信号**：代表消息的参数的取值是离散的。
    *   **码元**：在使用时间域的波形表示数字信号时，代表不同离散数值的基本波形。（二进制编码有0状态、1状态两种码元）。

### 2.2.2 有关信道的几个基本概念
*   **通信方向分类**：
    *   <font color="orange">单向通信（单工）</font>：只能有一个方向的通信。
    *   <font color="orange">双向交替通信（半双工）</font>：双方都可发送，但不能同时发送。
    *   <font color="orange">双向同时通信（全双工）</font>：双方可同时发送和接收。
*   **基带信号与调制**：
    *   <font color="orange">基带信号</font>：来自信源的信号，包含较多低频甚至直流成分。
    *   **基带调制（编码）**：仅改变波形。
        *   **不归零制**：正电平代表1，负代表0。
        *   **归零制**：正脉冲代表1，负代表0。
        *   <font color="orange">曼彻斯特编码</font>：周期中心向上跳变代表0，向下代表1（或反之）。
        *   <font color="orange">差分曼彻斯特编码</font>：位中心始终有跳变。位开始边界有跳变代表0，无跳变代表1。
    *   **带通调制**：使用载波，将频率范围搬移到较高频段，转为模拟信号。
        *   **调幅(AM)**、**调频(FM)**、**调相(PM)**。
        *   **正交振幅调制(QAM)**：多元制的振幅相位混合调制。
        *   📝 *示例*：若相位有12种，每种相位有1或2种振幅，总共16种组合（16个码元）。对应4 bit编码，数据传输率可提高4倍。

> ⚠️ **注意点**：不归零制没有自同步能力；而**曼彻斯特编码**和**差分曼彻斯特编码**具有<font color="orange">自同步能力</font>（能从波形本身提取时钟频率）。

### 2.2.3 信道的极限容量 🚀
实际信道受带宽、噪声、干扰和失真影响，会产生<font color="orange">码间串扰</font>（接收端信号波形失去清晰界限）。限制传输速率的两个因素：频率范围和信噪比。

**1. 奈氏准则（针对频率范围）**
在带宽为 $W$ (Hz) 的低通信道中，若不考虑噪声影响：
$$ \boxed{ 码元传输的最高速率 = 2W \text{ (码元/秒)} } $$

**2. 香农公式（针对信噪比）**
*   **信噪比(dB)** 计算公式：
    $$ \boxed{ \text{信噪比(dB)} = 10 \log_{10}(S/N) } $$
    *(示例：当 $S/N = 10$ 时为 10dB；$S/N = 1000$ 时为 30dB)*
*   **香农公式**：信道的极限信息传输速率 $C$：
    $$ \boxed{ C = W \log_2(1+S/N) \text{ (bit/s)} } $$
    其中 $W$ 为带宽，$S$ 为信号平均功率，$N$ 为高斯噪声功率。

> 🚨 **易错点解析：奈氏准则 vs 香农公式**
> *   **奈氏准则**：激励探索更先进的编码技术，让**每一个码元携带更多比特**（提高比特率）。
> *   **香农公式**：告诫在有噪声的实际信道上，不论编码多复杂，都**不可能突破极限**传输速率。

## 2.3 物理层下面的传输媒体 🔀

传输媒体是物理通路，分为两大类：
### 2.3.1 导引型传输媒体（固体媒体）
1.  <font color="orange">双绞线</font>：绞合度越高，传输率越高。分为无屏蔽(UTP)和屏蔽(STP)。在抗干扰能力上：U/FTP > F/UTP，而 F/FTP 是最好的。
2.  <font color="orange">同轴电缆</font>：抗干扰特性好，用于较高速率数据传输。
3.  <font color="orange">光缆/光纤</font>：利用光线全反射原理。
    *   <font color="orange">多模光纤</font>：多条不同角度光线传输，易失真，适合近距离。
    *   <font color="orange">单模光纤</font>：直径小至一个光波长，光线直线传播无多次反射，衰耗小，适合远距离（需昂贵激光器）。
    *   *光纤优点*：容量大、损耗小、抗雷电电磁干扰、保密性好、体积小重量轻。

### 2.3.2 非导引型传输媒体（无线传输）
*   **无线电微波通信**：主要直线传播。
    *   **微波接力**：利用天线塔（约100公里间距）中继。
    *   **卫星通信**：容量大、距离远。⚠️ *注意*：同步卫星传播时延较大（250~300ms），但“传播时延大”不等于“传送数据的时延大”。低轨道卫星系统（如SpaceX星链）正在普及。
*   **物理现象**：<font color="orange">多径效应</font>（信号经多次反射导致失真）。
*   **频段管理**：大部分需许可证，但 <font color="orange">ISM 频段</font>（如2.4GHz, 5.8GHz等）可以自由使用（用于无线局域网）。

## 2.4 信道复用技术 🛤️

复用允许用户使用一个<font color="orange">共享信道</font>进行通信。
1.  **频分复用 (FDM)**：将带宽分为多份，用户自始至终占用固定频带。所有用户在**相同时间**占用**不同频率**。
2.  **时分复用 (TDM)**：将时间划分为等长的TDM帧。用户在不同时间占用**相同频带**，呈周期性。易导致信道利用率不高。
3.  **统计时分复用 (STDM)**：按需动态分配时隙，不固定分配，提高线路利用率。
4.  **波分复用 (WDM)**：即光的频分复用。一根光纤同时传输多个波长（如1310nm, 1550nm）。
5.  <font color="orange">码分复用 (CDM) / 码分多址 (CDMA)</font>：
    *   **原理**：同时间、同频带通信。每位时间划分为 $m$ 个短间隔（码片）。
    *   每个站被指派一个**唯一且互相正交**的 $m$ bit 码片序列。
    *   发送 1：发原序列；发送 0：发反码序列。
    *   **正交特性公式**（向量 $S$ 和 $T$ 的规格化内积为 0）：
        $$ \boxed{ S \cdot T \equiv \frac{1}{m} \sum_{i=1}^{m} S_i T_i = 0 } $$
        *自身内积为 1，与自身反码内积为 -1。*

📝 **CDMA示例题目**：
*   假设 S 站的 8 bit 码片序列是 `00011011`。
*   为了计算，将0映射为-1，1映射为+1：$S = (-1, -1, -1, +1, +1, -1, +1, +1)$
*   发送 1 时，发送序列：`(-1 -1 -1 +1 +1 -1 +1 +1)`
*   发送 0 时，发送反码：`(+1 +1 +1 -1 -1 +1 -1 -1)`

## 2.5 数字传输系统 💻

*   **早期缺点**：速率标准不统一（北美的 T1 vs 欧洲的 E1），主要采用准同步方式，复用麻烦。
*   **现代标准**：
    *   美国标准：<font color="orange">SONET</font> (同步光纤网)，基础速率 51.84 Mbit/s (OC-1)。
    *   国际标准：<font color="orange">SDH</font> (同步数字系列)，基础速率 155.52 Mbit/s (STM-1)。
    *   *对应关系*：`STM-1` 相当于 `OC-3`。

## 2.6 宽带接入技术 🏠

分为有线和无线两大类。
1.  <font color="orange">ADSL 技术（非对称数字用户线）</font>：
    *   改造现有模拟电话用户线。低端频谱给电话，高端频谱给上网。
    *   **非对称**：下行带宽（ISP到用户）远大于上行带宽（用户到ISP）。
    *   采用 **DMT 调制技术**（使用 FDM 方法）。
    *   组成：DSLAM（接入复用器）、用户线、分离器、ATU-R（调制解调器）。
    *   ⚠️ *注意*：ADSL **不适合企业**，因为企业通常需要发送大量数据（需要大上行）。
2.  <font color="orange">HFC 网（光纤同轴混合网）</font>：
    *   基于有线电视网 CATV 改造。主干为光纤，入户为同轴电缆。
    *   设备：**机顶盒**（接电视接收数字信号）和 **电缆调制解调器 (Cable Modem)**（接计算机上网，需解决共享信道冲突）。
3.  <font color="orange">FTTx 技术（光纤接入）</font>：
    *   FTTH (到户), FTTB (到大楼), FTTC (到路边) 等。
    *   <font color="orange">光配线网 (ODN)</font> 位于光纤干线和用户间。无源的光配线网常称为 **PON**。
    *   PON 采用 **WDM**，上下行波长不同。
    *   主要流派：**EPON** (以太网协议，成本低) 和 **GPON** (吉比特，采用GEM封装，QoS好，成本高)。

## 📚 本章学习总结

本章深入探讨了计算机网络最底层的运作原理——物理层。学习的核心重点在于：
1.  **明确物理层的职责**：不在于具体的线缆材质，而在于定义四大接口特性，保证比特流的透明传输。
2.  **掌握数据通信的极限理论**：深刻理解**奈氏准则**（解决码间串扰）和**香农公式**（应对信道噪声）的区别与计算。
3.  **理解信号的调制与复用**：熟悉基带调制（曼彻斯特等）与带通调制（QAM等）；熟练掌握 FDM、TDM、STDM、WDM 及 **CDMA 的正交计算原理**。
4.  **了解前沿物理层技术**：理解从双绞线到光纤的演进（多模与单模），以及 ADSL、HFC 和 FTTx（EPON/GPON）等现代宽带接入技术的特点及适用场景。
