## 容易混淆的易错点回顾（虽未出错，考前再看一眼）

| 易混淆点 | 记忆方法 |
|---------|---------|
| 虚电路 vs 数据报 | 虚电路=网络保证可靠；数据报=主机保证可靠 |
| A/B/C/D 类开头 | 0 / 10 / 110 / 1110 |
| B 类范围 128~191 | B = Between A and C |
| IP 检验和只验首部 | IP 像快递单——只验单子有没有破，不验货物 |
| 片偏移单位 8 字节 | 分片大小必须是 8 的整数倍 |
| DF=1 超 MTU → 丢弃+ICMP | 不是分片，是直接丢 |
| ICMP 是 IP 层协议 | 虽封装在 IP 中，但属于网络层 |
| PING 不通运输层 | 应用层直接调网络层，没有 TCP/UDP |
| 不应发差错报文的情况 | 自身是差错/分片后续/多播/特殊地址 |
| RIP 距离 16=不可达 | 超过 15 跳就是 16=死亡 |
| RIP=UDP, OSPF=IP(89), BGP=TCP(179) | RIP 用 UDP (简单), OSPF 用裸 IP (高效), BGP 用 TCP (可靠) |
| RIP=距离向量, OSPF=链路状态+洪泛, BGP=路径向量 | 三个协议三种算法 |
| BGP 防环：AS-PATH | 看到自己的 AS 号就删 |
| NAT 出境换源 IP，入境换目的 IP | 把自己藏起来/把对方找回来 |
| NAPT 额外用端口号 | 多个内部 IP 复用一个全球 IP |
