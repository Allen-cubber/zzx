华南理工大学《数字信号处理II》深度复习报告与全真模拟考试分析 1. 执行摘要与考试策略分析 数字信号处理（Digital Signal Processing, DSP）作为电子信息类专业的核心课程，其理论体系严谨，数学推导繁复，是连接模拟世界与数字世界的桥梁。针对华南理工大学（SCUT）《数字信号处理II》课程的期末复习需求，本报告旨在提供一份详尽的、百科全书式的复习指南与全真模拟试卷。本报告不仅仅是一份试题集，更是一次对DSP核心知识体系的深度重构与剖析。 通过对2016年至2023年间多份华南理工大学期末试卷（包括2023期中、2023期末B卷、2022期中、2017A/B卷、2016B卷等）的详尽数据挖掘与文本分析，我们识别出了该课程考核的底层逻辑与高频考点。分析显示，试题设计高度强调理论与工程实践的结合，特别是离散时间系统的时频域分析、利用双线性变换法设计IIR滤波器、窗函数法设计FIR滤波器以及离散傅里叶变换（DFT）的快速算法与性质应用。 本报告长达数万字，将分为理论深度复盘、全真模拟试卷、以及超详尽的试题解析与考点延伸三个主要部分。我们在编写过程中严格遵循“高难度、全覆盖、重细节”的原则，力求模拟试卷的难度系数与往年真题（如、、）保持高度一致，甚至在某些易错环节（如频率预畸变、线性相位条件判断）进行了针对性的强化。 2. 核心理论体系深度复盘 在进入模拟考试之前，必须对支撑解题的理论框架进行极其细致的梳理。这一部分将超越教科书式的简单罗列，深入到公式背后的物理意义与数学推导细节，帮助考生建立起稳固的知识图谱。 2.1 离散时间信号与系统的时域分析 离散时间信号处理的基石在于对线性移不变（LTI）系统的深刻理解。LTI系统由其单位脉冲响应 $h[n]$ 唯一表征，其输入输出关系通过线性卷积和定义： $$y[n] = x[n] * h[n] = \sum_{k=-\infty}^{\infty} x[k] h[n-k]$$ 在往年试题中（如中的混响消除问题），经常考察利用差分方程描述系统行为的能力。常系数线性差分方程（LCCDE）的一般形式为： $$\sum_{k=0}^{N} a_k y[n-k] = \sum_{r=0}^{M} b_r x[n-r]$$ 对此类方程的求解通常涉及Z变换。考生必须熟练掌握系统因果性与稳定性的判断准则： 因果性：$h[n] = 0, \forall n < 0$。在Z域中，意味着收敛域（ROC）为某圆外区域且包含无穷远处。 稳定性：系统单位脉冲响应绝对可和，即 $\sum |h[n]| < \infty$。在Z域中，这等价于收敛域包含单位圆（$|z|=1$）。 深度洞察：仅仅记住结论是不够的。在复习中应注意到，对于有理系统函数 $H(z)$，系统的稳定性直接取决于极点的位置。对于因果系统，所有极点必须严格位于单位圆内。往年试题（如）常设置陷阱，给出非因果系统的差分方程，要求判断稳定性，此时必须先确定ROC的形态。 2.2 Z变换与频域分析 Z变换将离散序列映射到复频域，是分析离散系统瞬态响应和稳态特性的核心工具。 $$X(z) = \sum_{n=-\infty}^{\infty} x[n] z^{-n}$$ 在华工的考试中，Z变换的考察重点往往集中在以下几个方面： 收敛域（ROC）的确定：ROC的形状（圆内、圆外、圆环）决定了信号的时域特性（左边序列、右边序列、双边序列）。特别是当系统函数 $H(z)$ 给定时，必须结合因果性条件才能唯一确定ROC 。 逆Z变换：部分分式展开法是解题的必经之路。考生需熟练处理重极点情况以及共轭复根情况，能够快速识别常见变换对（如 $a^n u[n] \leftrightarrow \frac{1}{1-az^{-1}}$）。 几何评估法：通过零极点矢量图定性分析系统的频率响应。例如，极点靠近单位圆意味着该频率处的幅度响应有峰值，零点在单位圆上则意味着该频率完全被抑制（陷波）。 2.3 离散傅里叶变换（DFT）及其性质 DFT是数字信号处理中唯一能在计算机上直接实现的变换，其重要性不言而喻。 ...
