🗺️ 全书总框架#

本课程围绕无线电通信系统的信号处理链路展开，核心主题只有六个字：

放大、振荡、变换

信号产生 ──→ 信号放大 ──→ 信号变换 ──→ 信号发射
（振荡器）    （小信号/功放） （调制/混频/倍频）  （天线）
                                        │
                                        ▼
信号还原 ←── 信号选择 ←── 信号变换 ←── 信号接收
（解调/检波） （选频网络） （混频/变频）  （天线）

八章对应关系：

章节主题在全链路中的位置
第2章 选频网络频率选择与滤波贯穿始终（每一级都要用）
第3章 小信号放大器高频小信号放大接收机前级（RF/IF）
第5章 高频功率放大器高频大信号放大发射机末级
第6章 正弦波振荡器载波/本振信号产生发射载波源 + 接收本振源
第4章 非线性电路与变频器频率变换（混频/倍频）超外差接收机核心
第7章 振幅调制与解调AM/DSB/SSB 调制与检波发射调制 + 接收解调
第8章 角度调制与解调FM/PM 调制与鉴频发射调制 + 接收解调

📖 第1章 绪论#

核心考点#

1. 无线电发射机基本流程：

高频振荡器 → 高频功放 → 调制器 → 天线
                              ↑
声音 → 话筒 → 音频放大器 ────┘

2. 超外差接收机（最核心）：

天线 → 高频放大(RF) → 混频器 → 中频放大(IF) → 检波器 → 低频放大(AF) → 扬声器
                          ↑
                    本地振荡器(LO)
  • 核心部件：混频器
  • 核心作用（外差作用）：将不同载波频率统一变换为固定中频 (IF)
  • 优点：提高灵敏度和邻道选择性
  • 包络形状在混频过程中保持不变

3. 无线电波三种传播方式：

方式机制频率应用
地面波沿地球表面绕射越低越好长波/中波广播
天波电离层折射/反射短波短波广播通信
空间波视距直线传播超短波及以上电视/微波/卫星

易错点：短波靠天波（电离层），不是空间波；电视广播靠空间波，不是天波。

4. 调频广播频段：我国 FM = 87.5 ~ 108.0 MHz

📖 第2章 选频网络#

🔑 串联谐振 vs 并联谐振（核心对比）#

特性串联谐振并联谐振
谐振时阻抗最小（= R）最大（= Rₚ）
空载 Q₀ω₀L / RRₚ / ω₀L
谐振频率$f_0 = \frac{1}{2\pi\sqrt{LC}}$$f_0 = \frac{1}{2\pi\sqrt{LC_\Sigma}}$
适用源内阻 Rₛ很小（恒压源）很大（恒流源）
适用负载 Rₗ不大较大

一句话：串联要求源和负载都小，并联要求源和负载都大——否则 Q 值会被拉低。

信号源与负载接入的三大影响（必考）#

  1. Q 值下降 → 选择性变差，通频带加宽
  2. 功率传输下降 → 需要阻抗匹配
  3. 谐振频率偏移 → $C_\Sigma = C + C_s + C_L$ 变化

串并联阻抗等效互换#

转换方向电阻电抗
串联 → 并联$R_p = R_s(1+Q_s^2)$$X_p \approx X_s$ (高 Q)
并联 → 串联$R_s = R_p/(1+Q_p^2)$$X_s \approx X_p$ (高 Q)

高 Q 近似 (Q » 1)：$R_p \approx Q^2 R_s$，$X_p \approx X_s$

阻抗变换与接入系数 p#

抽头接入核心公式： $$R_{\text{高}} = \frac{R_{\text{低}}}{p^2}, \quad R_{\text{低}} = p^2 R_{\text{高}}$$

  • 电容抽头：$p = \dfrac{C_1}{C_1 + C_2}$
  • 变压器耦合：$p = N_2/N_1$，$R_L’ = n^2 R_L = R_L/p^2$
  • p 越小 → 变换后阻抗越大 → 对回路 Q 值影响越小

耦合回路（重点）#

耦合因子：$\eta = KQ$（衡量耦合强弱与回路损耗的相对关系）

η 范围名称响应曲线特征
η < 1弱耦合（欠耦合）单峰峰值 < 1，曲线尖锐
η = 1临界耦合单峰，峰值 = 1最平坦顶部，最宽单峰带宽
η > 1强耦合（过耦合）双峰（驼峰）峰值 = 1，$\xi_{peak} = \pm\sqrt{\eta^2-1}$

反射阻抗（互感耦合）：

  • $Z_{f1} = \frac{(\omega M)^2}{Z_{22}}$
  • 反射电阻 $R_{f1}$ 永远为正
  • 反射电抗 $X_{f1}$ 性质与次级 $X_{22}$ 相反

📖 第3章 高频小信号放大器#

核心概念#

1. 与低频放大器的主要区别：负载采用调谐回路（选频放大）

2. 主要性能指标：

指标含义
电压增益 $A_v$放大能力
通频带 $BW_{0.7}$增益下降 3dB（$1/\sqrt{2}$ 倍）的带宽
矩形系数 $K_{r0.1}$$BW_{0.1} / BW_{0.7}$，越接近 1 选择性越好
噪声系数 $NF$多级放大器主要取决于第一级

两种等效模型#

混合 π 模型 vs y 参数模型：

模型核心参数适用场景
混合 π$g_m$, $C_{b’e}$, $C_{b’c}$物理分析
y 参数$y_{fe}$ (正向传输导纳)高频小信号计算
  • $g_m$ = 晶体管跨导：$i_c = g_m u_{b’e}$
  • $C_{b’c}$ = 密勒电容，是限制高频增益的关键
  • $y_{fe}$ 的实部 $g_{fe}$ 决定放大能力，低频时 $g_{fe} \approx g_m$

单调谐放大器#

  • 谐振时电压增益最大：$A_{v0} = \dfrac{|y_{fe}|}{G_\Sigma}$
  • 有载 $Q_L = \dfrac{\omega_0 C_\Sigma}{G_\Sigma}$
  • 增益带宽积：$|A_{v0} \cdot BW|$ 为常数（增益和带宽互相制约）

📖 第4章 非线性电路、时变参量电路和变频器#

三种电路工作状态#

类型特点代表应用
线性电路叠加定理成立，不产生新频率小信号放大
非线性电路非线性元件 + 选频网络调制、解调、混频
时变参量电路参数随时间周期性变化混频器（“强本振弱信号”）

幂级数分析法#

非线性特性展开：$i = a_0 + a_1 v + a_2 v^2 + a_3 v^3 + \dots$

核心结论：二次方项 $a_2 v^2$ 产生乘积项——这是调制、混频的数学基础。

两个频率 $\omega_1, \omega_2$ 通过平方项产生：

  • 直流、$2\omega_1$、$2\omega_2$（倍频）
  • $\omega_1 + \omega_2$（和频）、$\omega_1 - \omega_2$（差频）→ 混频

混频器（超外差接收的核心）#

基本原理： $$f_I = f_o - f_s \quad \text{或} \quad f_I = f_o + f_s$$

关键指标：

概念含义公式
变频跨导 $g_c$输入高频电压 → 输出中频电流的能力$g_c = I_I / V_s \approx g_1 / 2$
时变跨导 $g(t)$本振信号周期性调制后的跨导$g(t)$ 的基波分量 $g_1$ 决定 $g_c$

混频干扰（重点简答）：

干扰类型产生条件特征
组合频率干扰$pf_o \pm qf_s\approx f_I$自身频率组合
副波道干扰外来干扰信号参与混频中频干扰、镜像干扰最常见
交叉调制干扰干扰信号被调制的包络转移到有用信号上与有用信号频率无关
互相调制干扰两个干扰信号非线性作用产生新频率落入中频通带内
阻塞干扰强干扰使放大器进入非线性区增益下降甚至阻塞

必记：镜像频率 $f_{image} = f_o + f_I = f_s + 2f_I$（上变频时）

📖 第5章 高频功率放大器#

工作状态分类#

状态导通角效率 η应用
甲类 (A)360°~50%小信号线性放大
乙类 (B)180°~78.5%推挽功放
甲乙类 (AB)180°~360°介于之间音频功放
丙类 (C)< 180°> 78.5%（可达90%+）高频功放主用
丁类 (D)开关接近 100%高效功放

丙类功放核心概念#

折线分析法（核心方法）：

  • 将晶体管转移特性 $i_c$-$u_{BE}$ 用折线近似
  • 导通角 $\theta$ 由偏置电压 $V_{BB}$ 决定：$\cos\theta = \dfrac{V_{BZ} - V_{BB}}{U_{bm}}$

三种工作状态（必考）：

状态负载线斜率$U_{cm}$ 变化特点
欠压较缓$U_{cm}$ 小$I_{cm1}$ 基本恒定，输出功率随 $R_L$ 增大而增大
临界最优—输出功率最大，效率较高（理想工作点）
过压较陡$U_{cm}$ 大出现凹陷，输出电压平坦，效率高但波形差

集电极调幅在过压区，基极调幅在欠压区。

动态跨导 $g_d$#

$$g_d = -g_c \frac{U_{bm}}{U_{cm}}$$

  • 是一个负值，反映输出电压和电流的动态关系
  • 斜率决定工作状态

📖 第6章 正弦波振荡器#

振荡条件（核心）#

条件表达式
振幅平衡（维持振荡）$AF= 1$
相位平衡$\sum \varphi = 2n\pi$
起振条件$AF> 1$

反馈型振荡器类型#

电路类型反馈方式要点
变压器耦合互感反馈通过同名端确定相位
电感三点式（哈特莱）电感分压$L_1, L_2$ 中间抽头接射极
电容三点式（考毕兹）电容分压$C_1, C_2$ 中间接射极
克拉泼考毕兹 + 串联电容提高频率稳定度
西勒克拉泼改进频率调节范围更宽
晶体振荡器石英晶体最高频率稳定度

石英晶体的两个谐振频率#

频率含义关系
$f_s$（串联谐振频率）$L_q C_q$ 串联谐振工作于串联型晶体振荡器
$f_p$（并联谐振频率）$L_q C_q C_0$ 并联谐振工作于并联型晶体振荡器（Pierce 等）

$f_p$ 略高于 $f_s$，两者非常接近。晶体工作在 $f_s$ ~ $f_p$ 之间呈感性。

📖 第7章 振幅调制与解调#

三种调幅信号对比（必考）#

特性AMDSB-SCSSB
表达式$V_0(1+m_a\cos\Omega t)\cos\omega_0 t$$kV_\Omega V_0\cos\Omega t\cos\omega_0 t$只取一个边带
频谱载波 + 上下边带只有上下边带（无载波）只有一个边带
带宽$2F_{max}$$2F_{max}$$F_{max}$
功率效率≤ 33.3%（$m_a$=1 时）100%100%
调制方式高电平/低电平低电平低电平
解调方式包络检波同步检波同步检波
优点接收简单节省功率最省功率和带宽
缺点功率效率低不能用包络检波系统复杂

调制度 $m_a$#

$$m_a = \frac{V_\Omega}{V_0}$$

  • $m_a = 1$：100% 调制（临界），最大不失真
  • $m_a > 1$：过调制，包络失真，接收端无法正确解调
  • $m_a < 1$：欠调制，功率利用率低

包络检波（AM 信号解调）#

核心结构：二极管 + RC 低通网络

RC 参数约束（设 $T_0$ 为载波周期）：

$$T_0 \ll RC \ll \frac{1}{\Omega_{max}}$$

RC 不能太小（否则高频纹波大），不能太大（否则惰性失真）。

四种失真（必考）：

失真原因避免条件
频率失真线性元件引起C 不影响上限，$C_c$ 不影响下限
非线性失真$V_{im}$ 或 R 过小大信号检波
惰性失真（对角线切割）RC 太大，放电跟不上包络下降$RC\Omega_{max} \leq \frac{\sqrt{1-m_a^2}}{m_a}$
负峰切割失真（底部切割）交直流负载不等（下一级 $r_{i2}$ 影响）$R_\Omega / R > m_a$

等效输入电阻：$R_{id} \approx R_L / 2$

同步检波（DSB/SSB 解调）#

  • 必须恢复载波，且与原载波同频同相
  • 频差 → 差拍振荡（振幅失真）
  • 相差 → 输出衰减 $\cos\varphi$，$\varphi = \pi/2$ 时无输出
  • 无失真关键：同步

📖 第8章 角度调制与解调#

FM vs PM 核心对比（必考）#

FM（调频）PM（调相）
调制信号控制瞬时频率瞬时相位
瞬时频率$\omega(t) = \omega_0 + k_f v_\Omega(t)$$\omega(t) = \omega_0 + k_p \frac{dv_\Omega}{dt}$
瞬时相位$\theta(t) = \omega_0 t + k_f \int v_\Omega dt$$\theta(t) = \omega_0 t + k_p v_\Omega(t)$
调制指数$m_f = k_f V_\Omega / \Omega$（与 Ω 成反比）$m_p = k_p V_\Omega$（与 Ω 无关）
最大频偏$\Delta\omega_f = k_f V_\Omega$（与 Ω 无关）$\Delta\omega_p = m_p\Omega$（与 Ω 成正比）
带宽特点恒定带宽调制随 Ω 增大而增大

核心区别口诀：FM 频偏不变、$m_f$ 随 Ω 变；PM 频偏随 Ω 变、$m_p$ 不变。

卡森公式（带宽估算）#

$$BW = 2(m_f + 1)F = 2(\Delta f_m + F)$$

  • 窄带 FM (NBFM)：$m_f < 1$ → $BW \approx 2F$
  • 宽带 FM (WBFM)：$m_f \gg 1$ → $BW \approx 2\Delta f_m$

调频波频谱特性#

  • 单音调制时：载波 + 无数对边带分量（与 AM 不同！）
  • 振幅由贝塞尔函数 $J_n(m_f)$ 决定
  • 总功率恒定 = 未调制载波功率 = $V_0^2 / 2R$
  • 改变 $m_f$ 只是功率重新分配，不影响总功率

调频方法#

方法原理优点缺点
直接调频调制信号直接控制振荡频率（变容二极管）频偏大，线路简单中心频率稳定度低
间接调频积分 → 调相 = 调频中心频率稳定（可用晶振）频偏较小

变容二极管调频#

$$C_j = \frac{C_{j0}}{(1 + V_R/V_D)^\gamma}$$

  • 理想线性调制条件：$\gamma = 2$
  • 一般突变结 $\gamma = 1/2$，超突变结 $\gamma = 1\sim 5$
  • 小频偏下，$\gamma \neq 2$ 也可近似线性

鉴频器对比#

相位鉴频器比例鉴频器
对幅度变化非常敏感不敏感（有动态限幅）
前级限幅器必须加不需要
输出电压较大约一半
核心改动—一个二极管反接 + 并联大电容
输出成分FM + AM 成分仅 FM 成分

比例鉴频器的限幅原理：大电容使 $V_{d1} + V_{d2}$ 保持恒定，动态调节 Q 值抵消幅度变化。

🔧 五种"跨导"总结（全书易混淆概念）#

跨导类型章节含义关键公式
晶体管跨导 $g_m$Ch3基射电压对集电极电流的控制$i_c = g_m u_{b’e}$
正向传输导纳 $g_{fe}$Ch3y 参数，输入电压对输出电流的同相控制低频时 $g_{fe} \approx g_m$
时变跨导 $g(t)$Ch4本振信号周期性调制后的跨导傅里叶级数分析混频
变频跨导 $g_c$Ch4输入高频电压 → 输出中频电流$g_c \approx g_1/2$
功放跨导 $g_c$（折线法）Ch5转移特性线性化后的斜率$i_c = g_c(u_{BE} - V_{BZ})$
动态跨导 $g_d$Ch5动态负载线的斜率（负值）$g_d = -g_c U_{bm}/U_{cm}$
鉴频跨导 $S_g$Ch8频率偏移 → 电路跨导变化比例$S_g = d(g_m)/d(f)$

📝 易混淆概念速查#

概念对区别
混频 vs 倍频混频是两个频率差/和 → 新频率；倍频是一个频率 × 整数
调制 vs 混频调制：低频信息搬到高频载波；混频：已调波搬到中频（包络不变）
包络检波 vs 同步检波包络检波用于 AM；同步检波用于 DSB/SSB（需恢复载波）
惰性失真 vs 负峰切割失真惰性 = RC 太大，放电赶不上；负峰切割 = 交直流负载不等
FM vs PM 调制指数FM 和 Ω 成反比；PM 和 Ω 无关
相位鉴频器 vs 比例鉴频器比例 = 改进型，有了自限幅
串联谐振 vs 并联谐振串联：源小负载小；并联：源大负载大
Q₀ vs Q_LQ₀ 是空载品质因数（回路自身），Q_L 是有载（接入源和负载后）

🎯 全书六个核心公式#

  1. 谐振频率：$f_0 = \dfrac{1}{2\pi\sqrt{LC}}$
  2. 接入系数阻抗变换：$R_{\text{高}} = R_{\text{低}} / p^2$
  3. 耦合因子：$\eta = KQ$（决定双调谐响应形状）
  4. 变频跨导：$g_c \approx g_1 / 2$（混频器核心指标）
  5. 调制指数：$m_a = V_\Omega / V_0$（AM），$m_f = \Delta\omega_f / \Omega$（FM）
  6. 卡森带宽：$BW = 2(m_f + 1)F = 2(\Delta f_m + F)$
