1. 基本概念 📚#

1.1 定义与特点#

扩展频谱通信（Spread Spectrum Communication）具有以下三个核心特点：

  1. 带宽极大：信号占用的带宽远远大于发送信息所需的最小带宽（几乎与信息速率无关）。
  2. 独码扩频：发送端利用扩频码（通常是伪随机序列）进行扩频，该码与发送信息无关。
  3. 同步解扩：接收端利用与发送端同步的扩频码进行解扩，以恢复原始信号。
  • 扩频通信的优点

    • 极强的抗干扰能力

      • 这是最突出的优点。如果有人用窄带信号干扰你，在接收端进行“解扩”时，有用的扩频信号变回了窄带（能量集中），而干扰信号反而被扩散成了宽带（能量被分散，幅度大幅下降）。此过程获得的信噪比改善称为处理增益。

    • 低截获概率

      • 因为频谱被展宽，单位频率上的功率（功率谱密度）极低，甚至低于环境噪声。敌方或窃听者很难发现这里有信号在传输，以为只是背景噪声。这在军事上称为“隐蔽通信”。

    • 抗多径衰落

      • 在城市中，信号反射会形成多径效应。扩频信号（特别是 DSSS）具有很好的自相关特性，可以通过 Rake 接收机 将不同路径到达的微弱信号分离并合并起来，变害为利，增强接收效果。

    • 码分多址 (CDMA) 能力

      • 不同用户可以使用相同的频率、在同一时间发送信号，只要大家分配到的 PN 码（扩频码）是正交的（互不相关），接收端就能把它们区分开来。这极大地提高了频谱利用率。
  • 扩频通信的缺点

    • 占用频带过宽

      • 对于单个用户来说，它占用了远超实际需要的频谱资源。如果用户数量很少，频谱利用率其实是偏低的（但在多用户 CDMA 系统中可以弥补这一点）。

    • 系统复杂度高

      • 同步 (Synchronization) 是扩频通信最难的点。接收端必须生成一个与发送端完全一致（相位、频率都对齐）的伪随机码才能解扩。一旦同步失败，整个通信就断了。

    • “远近效应” (Near-Far Problem)

      • 特别是在 DSSS/CDMA 系统中，如果一个干扰源（或另一个用户）离接收机很近，信号很强，它可能会淹没掉远处的弱信号。因此，CDMA 系统必须拥有极其精确的功率控制技术（让所有手机到达基站的信号强度基本一致）。

1.2 理论基础：香农公式#

扩频通信的本质是通过增加带宽来换取信噪比的改善。根据香农（Shannon）公式： $$C = B \log_2 (1 + \frac{S}{N})$$

  • 在信息传输速率 $C$ 一定的情况下，可以通过增加带宽 $B$ 来降低对信噪比 $S/N$ 的要求。
  • 当带宽足够大时，甚至可以在信噪比非常低（信号被淹没在噪声中）的情况下实现通信，这是保密通信的基础。

1.3 系统模型#

  • 发送端：信源 $\rightarrow$ 编码调制 $\rightarrow$ 扩频（由扩频码控制） $\rightarrow$ 载波调制 $\rightarrow$ 发射。
  • 接收端：接收 $\rightarrow$ 混频 $\rightarrow$ 解扩（由本地同步扩频码控制） $\rightarrow$ 解调解码 $\rightarrow$ 信宿。

1.4 技术指标 📊#

  1. 处理增益 ($G_p$)：表明扩频系统对信噪比的改善程度。 $$\boxed{G_p = \frac{(S/N)_o}{(S/N)_i}}$$

    • $(S/N)_o$：输出信噪比
    • $(S/N)_i$：输入信噪比

  2. 干扰容限 ($M_j$)：接收机能承受的干扰信号比有用信号高出的分贝数。 $$\boxed{M_j(dB) = G_p - [L_s + (\frac{S}{N})_o]}$$ $L_s$：系统损耗

2. 伪随机序列 (PN码) 🔢#

2.1 特征#

  • 伪随机性：确定产生但貌似随机，周期尽可能长。
  • 相关性：具有良好的自相关（便于捕获跟踪）和互相关（便于多用户）特性。
  • 均衡性：0和1出现的概率各为 $1/2$。
  • 游程特性：连0或连1称为游程。长为1的游程占1/2，长为2的占1/4，以此类推。

2.2 m序列 (最长线性反馈移位寄存器序列)#

  • 产生：由 $n$ 级线性反馈移位寄存器产生。
  • 周期：$$\boxed{N = 2^n - 1}$$ (周期最大值)。
  • 充要条件：特征多项式 $f(x)$ 必须是本原多项式。
    • $f(x)$ 是既约的（不可分解）。
    • $f(x)$ 可整除 $x^m + 1$ ($m=2^n-1$)。

📝 示例题目：m序列生成#

已知：3级反馈移位寄存器，特征多项式 $f(x) = 1 + x + x^3$，初始状态 ${a_{-3}, a_{-2}, a_{-1}} = {1, 0, 0}$。 求解：产生的m序列。

解法1：利用反馈逻辑 由 $f(x) = 1 \cdot x^0 + 1 \cdot x^1 + 0 \cdot x^2 + 1 \cdot x^3$ 可知反馈逻辑为 $a_n = a_{n-1} \oplus a_{n-3}$。

  1. $a_0 = a_{-1} \oplus a_{-3} = 0 \oplus 1 = 1$
  2. $a_1 = a_0 \oplus a_{-2} = 1 \oplus 0 = 1$
  3. $a_2 = a_1 \oplus a_{-1} = 1 \oplus 0 = 1$
  4. $a_3 = a_2 \oplus a_0 = 1 \oplus 1 = 0$
  5. $a_4 = a_3 \oplus a_1 = 0 \oplus 1 = 1$
  6. $a_5 = a_4 \oplus a_2 = 1 \oplus 1 = 0$
  7. $a_6 = a_5 \oplus a_3 = 0 \oplus 0 = 0$ 结果：输出序列为 ${1, 1, 1, 0, 1, 0, 0}$，周期为 $2^3 - 1 = 7$。

2.3 m序列的性质#

  1. 均衡性：“1”的数量比“0”多一个。
  2. 移位相加性：$M_p + M_r = M_s$ (移位后模2加仍是该序列的移位)。
  3. 自相关函数：二值函数。
    • $j=0$ (同步) 时，值为 1。
    • $j \neq 0$ (不同步) 时，值为 $-1/N$ (接近0)。
  4. 缺点：序列数量太少（受本原多项式数量限制），不适合大容量多用户系统。

2.4 Gold序列#

  • 构成：由两个码长、速率相同的 m序列优选对 进行模2加得到。
  • 优点：序列数量远大于m序列。
  • 特性：自相关和互相关特性呈现为三值函数，不再是理想的二值。

3. 直接序列扩频 (DSSS) 📡#

3.1 原理 (以BPSK为例)#

  • 发送：信息信号 $d(t)$ 与高速率的扩频码 $c(t)$ 直接相乘。 $$s_c(t) = \sqrt{2P} d(t) c(t) \cos \omega_0 t$$
  • 接收：利用本地同步的 $c(t)$ 再次相乘。
    • 由于 $c(t) \cdot c(t) = 1$ (双极性码)，信号被还原。

3.2 抗干扰机理#

  1. 有用信号：在发端被扩频（宽带），在收端经解扩恢复为窄带信号。
  2. 干扰信号：在传输途中加入（窄带），在收端经过解扩处理（乘以扩频码），频谱被展宽，能量分散。
  3. 滤波：通过窄带带通滤波器（BPF），大部分被展宽的干扰能量被滤除，只有恢复后的有用信号通过。

4. 跳频技术 (FH) 🐸#

4.1 原理#

  • 载波频率受伪随机序列控制，在某个频率集上随机跳变。
  • 直扩 vs 跳频：
    • 直扩：直接用码序列处理基带信号。
    • 跳频：用码序列控制频率合成器。

4.2 跳频系统抗干扰#

  • 有用信号频率随伪随机码跳变。
  • 干扰信号（通常是固定频率或窄带）无法跟上跳变规律。
  • 在接收端混频后，干扰信号被搬移到中频频带以外，被滤除。

4.3 关键部件：频率合成器#

  1. 直接式：混频+分频。转换快，频率多，但谐波多。
  2. 间接式 (PLL)：锁相环+VCO。输出纯度高，但转换速度受环路限制。

4.4 跳频图案#

  • 指载波频率随时间变化的规律。
  • 要求：周期长、随机性好（频率出现概率均等）、保密性高、不发生碰撞。

5. 扩频系统的同步 🔄#

同步是扩频通信的关键，分为两步：

5.1 捕获 (Acquisition)#

  • 目标：消除大的相位误差（使误差小于一个码片），是“粗同步”。
  • 方法：
    • 滑动相关法：改变本地码时钟频率，使其与接收码产生相对滑动，检测相关峰。
      • 串行捕获：结构简单，但搜索时间长。
      • 并行捕获：多路相关器同时工作，速度快，但电路复杂。

5.2 跟踪 (Tracking)#

  • 目标：捕获成功后，消除剩余微小误差，保持高精度同步，是“细同步”。
  • 方法：
    • 延迟锁定环 (DLL)：利用超前及滞后半个码片的相关能量差作为误差信号。
      • 包络相关跟踪环路。
      • $\tau$-抖动跟踪环路（解决了包络环路两路参数不一致导致的问题）。

6. 扩频技术的应用 🌍#

6.1 军事应用#

  • 优势：抗干扰、低截获概率 (LPI)、保密性。
  • 组网：利用正交跳频图案实现多电台组网。

6.2 民用应用#

  1. CDMA (码分多址)：
    • 利用扩频码的正交性（低互相关），不同用户使用同一频率同时传输。
    • 提高系统容量和频谱利用率。
  2. WLAN/蓝牙：
    • 在ISM开放频段工作，利用扩频技术抵抗各种干扰。
  3. GPS/测距：
    • 利用扩频码的高分辨率特性（码片窄），通过精确测量信号传播延时来测距。
    • C/A码精度约1微秒，P(Y)码精度约0.1微秒。

7. 本章学习总结 🧠#

  • 核心思想：利用带宽换取信噪比 (Shannon定理)。
  • 两大主流技术：
    • DSSS (直扩)：码乘信号，抗干扰靠接收端解扩“分散干扰能量”。
    • FH (跳频)：码控频率，抗干扰靠“躲避”和接收端“滤除带外干扰”。
  • 关键资源：伪随机序列 (m序列、Gold序列)。m序列自相关好但数量少；Gold序列数量多但相关性是三值的。
  • 实现难点：同步技术（捕获与跟踪）。

⚠️ 易错点与注意点#

  1. 处理增益与干扰容限：不要混淆公式。处理增益只看带宽比（或信噪比改善比），干扰容限要考虑系统损耗和输出信噪比要求。
  2. 扩频后带宽：扩频后的信号带宽主要取决于扩频码的速率，而不是信息原本的速率。
  3. m序列周期：是 $2^n - 1$ 而不是 $2^n$（全0状态不能进入反馈循环）。
  4. 同步顺序：必须先捕获（粗调），再跟踪（微调），顺序不能反。
  5. DSSS解扩：只有当本地扩频码与接收信号的扩频码完全同步（相位对齐）时，相乘结果才为1，才能恢复信号；否则输出为类噪声信号。
