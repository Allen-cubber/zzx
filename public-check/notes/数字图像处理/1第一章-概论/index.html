🏷️ 1.1 基本概念#

一、 图像与数字图像的区别#

  • 图像：是二维或三维景物呈现在人心目中的影象 。

    • 特点：包含大量信息（“一幅图像顶得上千言万语”），形式涵盖照片、绘图、视频等 。人类外界信息获取中约75%以上源于视觉系统 。

    • 表示：连续图像采用连续函数表示为 f(x,y)，其中 x 和 y 代表空间坐标位置，f 表示坐标点上的某种性质数值（例如亮度），这三个变量均可取任意实数 。

  • 数字图像：为了便于计算机进行加工处理，将连续图像在坐标空间和性质空间上都进行离散化，这种离散化的图像即为数字图像，也是本课程的主要研究对象 。

    • 在计算机中，数字图像被分割为一个个“像素”，每个像素的灰度值使用整数来表示 。

    • 对于一幅拥有M行N列像素的数字图像，其像素灰度值可以用矩阵形式表示 ：

      $$\boxed{G=\begin{bmatrix}g_{11}&g_{12}&\dots&g_{1N}\g_{21}&g_{22}&\dots&g_{2N}\\vdots&\vdots&\ddots&\vdots\g_{M1}&g_{M2}&\dots&g_{MN}\end{bmatrix}}$$

💡 易错点提醒：

在计算图像存储数据量时，一定要分清**比特(bit)与字节(byte)**的区别（8 bit = 1 byte）。

  • 示例题目 1 (单通道灰度图)：一幅256 * 256分辨率的船只图像。

    数据量 = 256 * 256 * 8 = 524288 (bit) = 65536 (byte) 。

  • 示例题目 2 (彩色三通道图)：一幅512512分辨率的女性图像（含RGB三个通道，故乘以3）。

    数据量 = 512 * 512 * 8 * 3 = 6291456 (bit) = 786432 (byte) 。

二、 数字图像处理与图像工程#

  • 数字图像处理：指应用计算机来合成、变换已有的数字图像，从而产生新的效果并重新输出图像的过程 。其流程为：图像输入 -> 图像处理系统 -> 图像输出 。

  • 图像工程：是一门交叉学科，横跨数学、物理学、计算机科学等多门领域 。它自下而上可以分为三个主要层次：

    1. 图像处理：这是一个“从图像到图像”的过程，主要目的是对图像进行加工、改善视觉效果。它针对底层像素进行操作，面临的数据量大，但语义低 。

    2. 图像分析：这是一个“从图像到数据”的过程，主要目的是分割和提取感兴趣的目标，获得客观信息并建立描述。它以观察者为中心 。

    3. 图像理解：这属于高层操作，研究目标的性质与相互联系，得出对场景的理解解释。它针对高层的符号进行运算，面临的数据量小，但语义高 。

🕰️ 1.2 数字图像处理的起源及发展应用#

  • 20年代：技术萌芽。例如利用打印方法及穿孔纸带传输得到5个灰度级的图像 。

  • 60年代：1964年，美国利用计算机辅助（FFT算法）成功绘制出月球表面地图 。

  • 70年代：技术有较大发展，1976年出版了第一本图像处理专著 。

  • 80年代：开始由二维(2D)图像处理迈向三维(3D)图像处理，获得广泛应用 。

  • 90年代：以多媒体技术为代表，深入到人类生活的方方面面 。

  • 21世纪：彻底改变了人们的生活方式 。

🌐 1.3 数字图像处理的应用实例#

图像处理技术在整个电磁波谱中均有极大的应用场景 ：

  • 伽马射线：主要用于医学（如PET图像、骨骼扫描）和天文观测 。

  • X射线：医学（如胸透、CT）、工业领域（电路板检测）及天文学 。

  • 紫外线：平板印刷术、工业检测、显微镜方法、生物成像等 。

  • 可见光及红外线：广泛应用于天文、遥感、工业等方面 。日常实例包括指纹识别、纸币检测、车牌自动读取等 。

  • 微波波段：核心用于雷达成像（如探测山脉地形） 。

  • 无线电波段：主要用于医学（如核磁共振MRI图像）和天文学 。

应用领域总结：涉及遥感技术（资源调查、城市规划）、医学图像（超声波、CT、核磁共振）、工业领域（无损探伤、机器人视觉）、军事领域（目标跟踪、指纹识别）、文化艺术（文物修复）及气象、自动驾驶等多维度领域 。

🔬 1.4 研究内容和方法#

一、 五大系统模块#

一个基本的图像处理和分析系统由五个模块构成：图像采集系统、图像处理与分析（核心计算模块）、图像输出系统、图像存储系统以及通信系统 。

二、 六大核心研究内容#

  1. 图像增强：改善图像的主观质量（如灰度变换、直方图修正、平滑与锐化滤波等） 。

  2. 图像恢复（复原）：根据已知的图像退化原因（建立合理的退化模型），消除退化影响，力求产生出等价于理想系统获取的图像 。

  3. 图像压缩编码：由于图像数据量大、冗余多，通过有损或无损压缩算法减小数据体积（如预测编码、变换编码） 。

  4. 图像分割：按灰度、颜色、几何性质等，将目标物体从图像中分离出来，这是后续分析的基础步骤 。

  5. 目标表达和描述：将分割出的物体用新形式表达（边界或内部区域表达），并使用数学语言进行简练的描述建模（如结构描述符） 。

  6. 图像模式分类：基于提取的特征为目标划分指定类别。例如运用深度神经网络（经由卷积层提取特征、池化、最终通过全连接层输出分类结果） 。

三、 两种基本处理方法#

  1. 空间域处理：直接对图像的像素进行数学运算操作 。

  2. 变换域处理：将图像通过特定运算转换到其他数学域中处理。常用方法包括：傅里叶变换(DFT)、离散余弦变换(DCT)、离散小波变换(DWT)等 。

💡 1.5 技术特点#

  • 数字图像本身的特点：

    • 数据量极大：例如一张普通512 * 512大小的灰度图像约占260KB存储，而遥感和雷达卫星图像往往大得多 。

    • 高度相关性：像素之间存在很大关联，这意味着冗余信息极多，为数据压缩提供了空间 。

  • 数字图像处理的优点（相较于传统光学处理）：

    • 灵活性强：方便进行数字化拼贴、调整与任意修改 。

    • 精度高：分辨率不受限且计算机处理严谨方便 。

    • 再现性好：数据可永久稳定保存，不存在物理退化问题 。

🎓 本章学习总结#

本章作为《数字图像处理》的概论，带我们从宏观视角认识了数字图像的基础矩阵表达方式，并理清了图像处理（底层/像素）、图像分析（中层/目标）与图像理解（高层/符号）这三个容易混淆的概念层级。通过回顾电磁波成像在医疗、工业、遥感等领域的广泛应用，确立了这门技术的实用价值。最后明确了后续学习的主线：从获取图像开始，历经增强、恢复、压缩、分割直到最终的目标识别分类，并掌握空间域和变换域两大基本方法框架。
