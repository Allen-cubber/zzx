二、简答题 (每题10分)

  1. 简述8051单片机和8031单片机在构成最小系统时的主要区别。

    • 片内资源 (3分): 8051有片内ROM，8031无片内ROM。两者都有片内RAM。
    • EA引脚接法 (3分): 8051最小系统EA接高电平（访问片内ROM）。8031最小系统EA接低电平（强制访问片外ROM）。
    • 外部电路需求 (4分): 8051只需时钟和复位电路。8031除了时钟和复位电路，还必须外扩程序存储器（如EPROM），并需要地址锁存器（如74LS373）来分离P0口的地址/数据。

  2. MCS-51单片机的并行扩展总线是如何形成的？

    • 地址总线 (4分): 16位。高8位(A15-A8)由P2口直接提供。低8位(A7-A0)由P0口分时提供，在ALE信号控制下由外部地址锁存器锁存。
    • 数据总线 (3分): 8位双向。由P0口在地址信号锁存后分时提供。
    • 控制总线 (3分): 主要包括ALE（地址锁存允许，P2.6附近硬件产生）、$\overline{PSEN}$（程序存储器读选通，P2.7附近硬件产生）、$\overline{RD}$（外部数据存储器读，P3.7）、$\overline{WR}$（外部数据存储器写，P3.6）等信号。

  3. 比较MCS-51扩展外部程序存储器和扩展外部数据存储器在地址空间、访问指令和读写控制信号上的主要异同点。

    • 相同点 (2分): 都可以扩展最大64KB。
    • 不同点 (8分):
      • 地址空间 (2分): 程序存储器和数据存储器各有独立的64KB地址空间，它们的地址可以重叠，互不干扰。
      • 访问指令 (3分): 访问外部程序存储器（通常是查表）用MOVC指令。访问外部数据存储器用MOVX指令。CPU取指令是自动访问程序存储器空间。
      • 读写控制信号 (3分): 读外部程序存储器用$\overline{PSEN}$信号。读外部数据存储器用$\overline{RD}$信号，写外部数据存储器用$\overline{WR}$信号。

  4. 在进行I/O端口扩展时，为什么输出口通常用锁存器，而输入口通常用三态缓冲器？外部I/O端口和外部数据存储器的地址空间是如何管理的？

    • 原因 (6分):
      • 输出口用锁存器 (3分): CPU向输出口写数据时，数据在P0总线上只出现很短时间。锁存器（如74LS273）可以在$\overline{WR}$信号有效（并配合地址译码产生的锁存脉冲）时将P0口上的数据锁存下来并稳定输出给外部设备，直到下一次写入。
      • 输入口用三态缓冲器 (3分): 输入时，外部设备的数据需要被CPU读取。三态缓冲器（如74LS244）在$\overline{RD}$信号有效（并配合地址译码产生的选通信号）时，将外部设备的数据连接到P0总线上，CPU通过P0口读取。在不读该输入口时，三态缓冲器处于高阻态，不影响P0总线的其他操作。
    • 地址空间管理 (4分): 外部I/O端口和外部数据存储器采用统一编址的方式。它们共享同一个64KB的外部数据存储器地址空间，使用相同的访问指令 (MOVX) 和相同的读写控制信号 ($\overline{RD}$, $\overline{WR}$)。因此，在系统中，一个特定的地址只能分配给外部数据存储器或外部I/O端口之一，不能重叠。

三、分析与设计题 (共30分)

  1. 地址范围分析 (15分):

    • EPROM (2KB = $2^{11}$ Bytes, A0-A10) (7分): 片选条件：P2.7=0, P2.6=1, P2.5=0。其余高位(A15, A14, A13, A12, A11)无关且设为0。 地址格式： A15 A14 A13 A12 A11 | P2.7 P2.6 P2.5 | A10 ... A0 选通时： 0 0 0 0 0 | 0 1 0 | A10 ... A0 二进制表示： 0000 0010 0xxxxxxxxxxxB (x代表A10-A0) 起始地址 (A10-A0全0): 0000 0010 0000 0000 0000B = 0400H 结束地址 (A10-A0全1): 0000 0010 0111 1111 1111B = 0400H + 07FFH = 0BFFH 所以EPROM地址范围是：0400H ~ 0BFFH
    • SRAM (2KB = $2^{11}$ Bytes, A0-A10) (8分): 片选条件：P2.7=1, P2.6=0, P2.5=1。其余高位(A15, A14, A13, A12, A11)无关且设为0。 地址格式： A15 A14 A13 A12 A11 | P2.7 P2.6 P2.5 | A10 ... A0 选通时： 0 0 0 0 0 | 1 0 1 | A10 ... A0 二进制表示： 0000 0101 0xxxxxxxxxxxB 起始地址 (A10-A0全0): 0000 0101 0000 0000 0000B = 0A00H 结束地址 (A10-A0全1): 0000 0101 0111 1111 1111B = 0A00H + 07FFH = 11FFH 所以SRAM地址范围是：0A00H ~ 11FFH

  2. 电路连接设计简图 (15分):

    • EPROM (2732, 4KB = $2^{12}$ Bytes, A0-A11) 地址0000H-0FFFH:
      • 8031的P0口接74LS373的D0-D7输入，74LS373的Q0-Q7输出接EPROM的A0-A7。
      • 8031的ALE接74LS373的G（或LE,取决于具体锁存器型号）。74LS373的$\overline{OC}$接地。
      • 8031的P2.0-P2.3接EPROM的A8-A11。
      • EPROM片选$\overline{CE}$: 需要当A15-A12为0000B时有效。可以使用一个四输入与非门，将P2.4(A12), P2.5(A13), P2.6(A14), P2.7(A15)作为输入，与非门输出接EPROM的$\overline{CE}$。（简单化：如果只有这一个EPROM，可以直接将$\overline{CE}$接地，但题目要求地址范围，暗示有译码）
        • 更简单的译码：若系统只有这个EPROM，且地址不与其他冲突，可以直接将P2.7（作为A15）用作片选。如果A15=0选中，则地址范围是0000H-7FFFH，再由A0-A11决定内部4KB。但题目指定0000H-0FFFH，意味着A15-A12都应为0。最简单是用一个4输入与非门。如果允许不完全译码，可以用P2.7=0作为$\overline{CE}$，则实际地址会镜像。
      • 8031的$\overline{PSEN}$接EPROM的$\overline{OE}$（或$\overline{RD}$）。
      • 8031的P0口（在数据周期）接EPROM的D0-D7数据线。
      • 8031的EA接地。
    • 输出口 (74LS273) 地址8000H:
      • 8031的P0口接74LS273的D0-D7输入。
      • 74LS273的Q0-Q7作为输出口。
      • 输出口片选译码：地址8000H，即A15=1, A14-A0=0。
        • 译码逻辑：P2.7(A15)直接连接到一个与门的一个输入。其他地址线A14-A0（或部分关键位如A14,A13,A12）都为0时，译码电路输出一个低电平有效信号。例如，用一个多输入与非门，输入为P2.7的非、P2.6的非…P2.0的非（假设DPTR高字节设为80H）。更简单的是只用A15=P2.7=1作为主要片选条件。
        • 若只用A15=1（即P2.7=1）来片选，则该输出口地址范围是8000H-FFFFH。
        • 精确译码：要使地址为8000H，则A15=1, A14=0, …, A0=0。可以用一个16输入与门（或等效逻辑）产生片选，但这太复杂。通常是部分译码。例如，取P2.7=1 (A15), P2.6=0 (A14), P2.5=0 (A13), P2.4=0 (A12) 作为片选条件。
      • 锁存脉冲CLK：将译码产生的片选信号（低有效）与8031的$\overline{WR}$信号通过一个与非门（或或门后取反）连接到74LS273的CLK端（注意CLK是上升沿有效，所以$\overline{WR}$下降沿时数据稳定，$\overline{WR}$上升沿锁存）。
        • 即 CLK_273 = NOT(NOT(ChipSelect_8000H) OR WR)。 或者 CLK_273 = ChipSelect_8000H_ActiveHigh AND NOT(WR_pulsed)。
        • 更简单：$\overline{CE}_{273}$ = 片选译码(低有效)；CLK = $\overline{WR}$的上升沿 (可能需要反相器和边沿触发逻辑，或直接用$\overline{WR}$的反相信号，并确保数据建立时间)。最常见的是，译码信号作为使能，$\overline{WR}$产生锁存脉冲。
    • 评分点：
      • P0与373, EPROM, 273数据线连接正确 (3分)
      • ALE与373控制连接正确 (1分)
      • P2与EPROM高位地址连接正确 (2分)
      • $\overline{PSEN}$与EPROM读控制连接正确 (2分)
      • EA接地 (1分)
      • EPROM片选逻辑基本合理，能体现0000H-0FFFH范围 (2分)
      • 74LS273片选逻辑基本合理，能体现8000H地址 (2分)
      • $\overline{WR}$与273锁存控制连接基本合理 (2分)
