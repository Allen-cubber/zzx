8.1 概述 🌐#

  • 计算机控制系统通常包含：控制对象 ➡️ 传感器 ➡️ 模/数转换器 (A/D) ➡️ 计算机 ➡️ 数/模转换器 (D/A) ➡️ 执行机构 。

8.2 D/A转换器的电路结构和工作原理 🔌#

1. 权电阻网络 D/A转换器#

  • 原理：开关状态受输入数码控制，当 $d_i=1$ 时开关接到参考电压 $V_{REF}$ 上，当 $d_i=0$ 时开关接地 。权电阻网络使各支路电流与权电阻成反比 。

  • 输出电压公式：输出的模拟电压与输入的数字量 $D_n$ 成正比 。

    $$\boxed{v_o = -\frac{V_{REF}}{2^n}D_n}$$

    输出电压的范围为 $0 \sim -\frac{2^n-1}{2^n}V_{REF}$ 。

  • 优缺点：

    • 优点：电路结构简单，所用的电阻元件少 。

    • 缺点：不同电阻的阻值相差较大，输入数字量的位数越多，差别就越大，很难保证电阻的精确度 。

  • 💡 示例：8位二进制输入，输出为 0~25.5V。当输入最高位是“1”，其余是“0”（即 128）时，求输出电压 。

    • 先求基准电压：$V_{REF} = \frac{2^8}{2^8-1} \times 25.5V = -25.6V$ 。

    • 代入公式：$v_o = -\frac{-25.6}{2^8} \times 128 = 12.8V$ 。

2. 倒T型电阻网络 D/A转换器#

  • 原理：电阻网络中只有 $R$ 和 $2R$ 两种阻值的电阻，方便制作成集成电路 。无论开关打在哪一面，都相当于接到了“地”上，流过每个支路的电流始终不变 。

  • 输出电压公式：同上，与输入数字量成正比 。

    $$\boxed{v_o = -\frac{V_{REF}}{2^n}D_n}$$

  • 💡 示例：10位 R-2R 网络 DAC，输入 $(100)H$ 时，输出 $v_o=2V$。求 $V{REF}$ 和输入 $(200)_H$ 时的输出 。

    • $(100)H = 256$，由此算出 $V{REF} = -\frac{1024 \times 2}{256} = -8V$ 。

    • $(200)_H = 512$，代入得 $v_o = -\frac{-8}{2^{10}} \times 512 = 4V$ 。

3. 权电流型 D/A转换器#

  • 原理：利用一组恒流源构成“权”，每个支路电流的大小不再受开关内阻和压降的影响，降低了对开关电路的要求 。

  • 公式：

    $$\boxed{v_o = \frac{R_F V_{REF}}{R_R 2^n}D_n}$$

4. 具有双极性输出的 D/A转换器#

  • 原理：在求和放大器的输入端接入一个偏移电流，使输入最高位为1且其他各位为0时，输出 $v_o=0$ 。

  • 二进制补码数模转换要求：将输入的符号位（最高位）反相接入 D/A 转换器 。

  • 公式（加入偏移后）：

    $$\boxed{v_o = -\frac{V_{REF}}{2^n}(D_n - 2^{n-1})}$$

8.3 D/A转换器的转换精度与转换速度 ⏱️#

  • 分辨率（理论精度）：描述对输入微小量变化的敏感程度，常用数字量的位数表示（如 $\frac{1}{2^n-1}$）。

  • 转换误差（实际精度）：实际特性和理论特性之间的最大偏差，常用最低有效位的倍数表示（如 $\pm \frac{1}{2} LSB$）。总误差来源包括：基准电压偏离、运放零点漂移、模拟开关导通电阻压降等 。

  • 转换速度：用建立时间 $t_{set}$ 描述，即从输入数字量发生突变开始，到输出电压进入与稳态值相差 $\pm \frac{1}{2} LSB$ 范围以内所用的时间 。

8.4 A/D转换的基本原理 🔄#

模-数转换的基本步骤包括：

  1. 采样：将时间连续的模拟量转换为时间离散的模拟量 。

    • 📌 采样定理：为保证能恢复原信号，采样频率须满足 $\boxed{f_s \ge 2f_{i(max)}}$ 。

  2. 保持：给后续量化编码提供稳定值 。

  3. 量化：将采样电压表示为最小数量单位（$\Delta = 1 LSB$）的整数倍 。

    • ⚠️ 注意点：量化误差属于原理性误差，无法消除，位数越多误差越小 。

  4. 编码：将量化后的值转换为对应的数字代码 。

8.6 A/D转换器的电路结构和工作原理 ⚙️#

1. 并联比较型 A/D转换器#

  • 原理：通过分压电阻得到一系列比较电平，把 $0 \sim V_{REF}$ 划分为多个区间，分别接入比较器负端，输入电压与它们同时比较 。

  • 优缺点：

    • 优点：转换速度极快；兼有采样-保持功能 。

    • 缺点：需要大量比较器和触发器（n位输出需要 $2^n-1$ 个比较器）。

2. 逐次逼近型 A/D转换器#

  • 原理：采用反馈比较型电路。取一个数字量加到 D/A 转换器上产生模拟电压，与输入模拟电压比较。不相等则调整数字量，直到两者相等（类似“天平称重”）。每次从半程（最高位为1）开始比较 。

  • 特点：电路复杂度适中，速度适中（比计数型快得多），是最常用的类型。n位转换需要 $n+2$ 个时钟周期 。

3. 双积分型 A/D转换器（V-T型）#

  • 原理：属于间接 A/D 转换器。先将模拟电压转换为时间宽度信号，再在这个时间内对时钟脉冲计数 。

    • 第一阶段：对 $v_I$ 积分，时间 $T_1$ 固定 。

    • 第二阶段：对 $-V_{REF}$ 反向积分，直到 $v_o=0$，积分时间 $T_2$ 变化 。

  • 公式：$T_2$ 与输入电压成正比，即为中间量。

    $$\boxed{T_2 = \frac{T_1}{V_{REF}} v_I}$$

  • 优缺点：

    • 优点：转换结果与电路参数（$R$、$C$）及时钟周期无关，对元器件精度要求不高 。

    • 缺点：工作速度很慢 。

  • 💡 示例：$V_{REF}=-10V$, $n=10$, $f_c=10kHz$。输入 $v_I=5V$ 时求转换时间与输出 。

    • 转换时间：$T = (1 + \frac{v_I}{V_{REF}})T_1 = 2^{10} \times \frac{1}{10^4}(1+0.5) = 0.1536s$ 。

    • 数字量输出：$D = \frac{2^{10}}{10} \times 5 = 512 = (100000000)_B$ 。

4. V-F变换型 A/D转换器#

  • 原理：输出频率 $f_{out}$ 与输入电压成正比 。

  • 特点：抗干扰能力极强，常用于遥测、遥控系统 。

8.7 A/D 的精度与速度对比 📊#

  • 速度排名 ：

    1. 并联比较型：最快，$< 1\mu s$。

    2. 逐次逼近型：居中，$10 \sim 100 \mu s/$次。

    3. 双积分型 / V-T变换型：最慢，几十毫秒/次。

🌟 本章学习总结#

本章系统地介绍了模数（A/D）和数模（D/A）转换技术。

  • 对于 D/A转换，我们需要重点掌握权电阻网络和倒T形电阻网络的计算公式，并理解双极性偏移输出的原理 。

  • 对于 A/D转换，必须牢记其核心四步曲（采样、保持、量化、编码）和奈奎斯特采样定理 。在具体电路中，要清楚直接型（并联比较、逐次逼近）与间接型（双积分、V-F型）在速度与精度上的折中关系 。

🚨 易错点提醒： 1. 在计算双极性 D/A 时，务必注意偏移量的方向，牢记其数字输入需要视作补码并进行符号位反相操作 。 2. 量化误差是 A/D 转换器固有的原理误差，无论怎么改善工艺都无法消除，只能通过增加位数来减小 。

模块一：D/A 转换（数模）—— 死磕倒T型网络计算 🧮#

考试遇到 D/A 转换计算题，认准倒T型(R-2R)电阻网络，直接套用以下两个公式：

  • 1. 基础单极性输出公式： 输出的模拟电压 $v_o$ 与输入的十进制数字量 $D_n$ 成正比 。

    $$\boxed{v_o = -\frac{V_{REF}}{2^n}D_n}$$

    • 代入说明：$n$ 是题目给的位数（比如 8 位、10 位），$D_n$ 是输入的二进制数转换成的十进制值，$V_{REF}$ 是基准电压。

  • 2. 双极性（偏移）输出公式： 如果题目明确提到“双极性”或有“偏移电流”，说明它把补码输入的正负数转换成了正负极性输出的模拟电压 。

    $$\boxed{v_o = -\frac{V_{REF}}{2^n}(D_n - 2^{n-1})}$$

⚠️ 避坑指南：计算时千万别忘了公式最前面的负号！如果 $V_{REF}$ 是负值，负负得正，最后的输出电压才是正的。

模块二：A/D 转换（模数）—— 必考概念默写 📝#

这部分是填空题和选择题的重灾区，记住以下三点就能稳拿基础分：

  • 1. 转换四大步骤（按顺序死记）： 1、采样 ➡️ 2、保持 ➡️ 3、量化 ➡️ 4、编码 。

  • 2. 采样定理（奈奎斯特）： 为了保证能从采样信号恢复原来的被采样信号，采样频率必须满足 ：

    $$\boxed{f_s \ge 2f_{i(max)}}$$

  • 3. 速度排行榜：

    • 🥇 最快：并联比较型 A/D 转换器 。

    • 🥈 中等：逐次逼近型（逐位渐近型）A/D 转换器 。

    • 🥉 最慢：间接 A/D 转换器（如双积分型、V-T变换型） 。

模块三：A/D 转换主考电路对比 ⚖️#

这部分大概率出简答题或综合计算题，重点区分下面两种：

1. 逐次逼近型 A/D 转换器（最常用）#

  • 通俗理解：采用反馈比较型电路结构 ，就像用天平称重。从最大砝码（半程）开始试 ，如果 $v_o > v_I$，则去掉这个1；如果 $v_o < v_I$，则保留这个1 。

  • 硬核考点：$n$ 位输出的转换器，需要 $n+2$ 个时钟周期 。

2. 双积分型 A/D 转换器（V-T型）#

属于间接转换，先将模拟电压转换为时间宽度，再计数 。记住以下推导结论：

  • 求中间量时间 $T_2$： 第一阶段积分时间 $T_1$ 和基准电压 $-V_{REF}$ 都是常数，因此 $T_2$ 与输入电压 $v_I$ 成正比 。

    $$T_2 = \frac{T_1}{V_{REF}}v_I$$

  • 求总转换时间 $T$：

    $$T = T_1 + T_2$$

  • 求输出数字量 $D$：

    $$D = \frac{2^n}{V_{REF}}v_I$$
